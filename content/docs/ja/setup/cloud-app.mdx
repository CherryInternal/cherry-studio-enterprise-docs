## 🚀 クイックスタート

Cherry Studio Enterprise は Tencent Cloud Marketplace に登場しました。ワンクリック導入に対応し、複雑な設定なしでエンタープライズ AI プラットフォームを素早く体験できます。

<Callout type="warn">
現在、クラウドアプリのワンクリック導入は Tencent Cloud のみ対応しています。他のクラウドプロバイダーにも順次対応予定です。
</Callout>

**Tencent Cloud Marketplace リンク**: [https://app.cloud.tencent.com/detail/SPU_BHFHEJFDEE4965](https://app.cloud.tencent.com/detail/SPU_BHFHEJFDEE4965)

<Callout type="info">
Tencent Cloud Marketplace では「スタンダード版」と表示されますが、実際のエディションはご購入時に発行される **ライセンス** によって決まります。ライセンスは**プレミアム版**と**エクスプレス版**の両方に対応しており、具体的なエディションはオフライン契約の内容に基づきます。
</Callout>

## 📋 導入フロー

### ステップ 1: インストール開始

1. [Tencent Cloud Cherry Studio インストールページ](https://app.cloud.tencent.com/detail/SPU_BHFHEJFDEE4965) にアクセス
2. Tencent Cloud アカウントでログイン
3. **インストール** ボタンをクリックしてリソースを選択
4. 近いリージョンを選んで導入

<Callout type="info">
    選択したリージョンにサーバーがありませんか？<a href="#no-machine">対処方法を見る</a>
</Callout>

### ステップ 2: 導入パラメータの設定

システムがネットワーク環境を自動検出します。適切なネットワーク/セキュリティグループがある場合は次へ進みます。

<img src="/assets/images/zh/cloudappinstall/17579059506041.webp" alt="アプリ生成" />
### ステップ 3: 作成完了まで待機

アプリ作成には数分かかります。しばらくお待ちください。

<img src="/assets/images/zh/cloudappinstall/17579075417655.webp" alt="作成待機" />
### ステップ 4: アクセス入口を取得

作成完了後、アプリ設定ページからアクセス入口を取得します：

<img src="/assets/images/zh/cloudappinstall/17579075847068.webp" alt="アプリ設定入口" />
**管理コンソール入口（パブリック）** をクリックすると、Cherry Studio Enterprise 管理コンソールにアクセスできます。

<Callout type="info">
    管理コンソールにアクセスできませんか？セキュリティグループがリクエストを許可していない可能性があります。<a href="#set-acl">対処方法を見る</a>
</Callout>

### ステップ 5: 初回ログイン

アプリ設定ページに表示される **Cherry Studio Enterprise 管理者アカウント** でログインします：

- 初期ユーザー名：アプリ設定ページに表示
- 初期パスワード：アプリ設定ページに表示

<img src="/assets/images/zh/cloudappinstall/17579076770749.webp" alt="管理コンソール入口" style={{width: '80%'}} />
### ステップ 6: 既定パスワードの変更

**強く推奨**：初回ログイン後すぐに管理者パスワードを変更してください。

<img src="/assets/images/ja/setup/change-password-1_ja.webp" alt="ユーザー管理-パスワード変更入口" />
<img src="/assets/images/ja/setup/change-password-2_ja.webp" alt="パスワード変更ダイアログ" />
## ✅ 導入完了

- ユーザーとロール権限の作成

## ❓ よくある質問

### Q: 可用区を選んだときにマシン一覧が表示されません。 [#no-machine]

A: Tencent Cloud サポートに問い合わせ、利用可能なゾーンを確認してください。例えば広州 6/7 にリソースがあり、2 ゾーンにはない場合があります。利用可能なゾーンで新しい VPC とサブネットを作成してください。

<Callout type="info">
    <a href="#create-vpc">VPC とサブネットの作成方法</a>
</Callout>

### Q: 導入後にアクセスできません。

A: 以下を確認してください：

1. セキュリティグループのポート設定が正しいか（36700、36800、32811）
2. ネットワーク接続が正常か
3. 正しいアクセスアドレスとポートを使用しているか

<Callout type="info">
    <a href="#set-acl">セキュリティグループの設定方法</a>
</Callout>
