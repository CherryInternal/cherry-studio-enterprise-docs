# 最初のメッセージ送信

おめでとうございます！すべての設定が完了しました。最初のメッセージを送って動作を確認しましょう。

## 会話を開始

1. クライアントで **新しいチャット** をクリック
2. 利用可能な AI モデルを選択
3. 例として次のメッセージを入力：

```
こんにちは！自己紹介してください。
```

4. Enter を押して送信

## 成功の確認

AI から返信が届けば、設定は問題ありません。

## よくある問題

### サーバーに接続できない

- サーバーアドレスが正しいか確認
- サービスが起動しているか確認（`http://your-server:3670/health` にアクセス）
- ファイアウォール設定を確認

### 利用可能なモデルがない

- 管理者がプロバイダーを設定済みか確認
- ユーザーが正しいグループに割り当てられているか確認
- 管理者にグループのモデル権限を確認してもらう

### メッセージ送信時にエラー

- プロバイダーの API Key が有効か確認
- モデルのクォータが不足していないか確認
- 管理コンソールの呼び出しログを確認
