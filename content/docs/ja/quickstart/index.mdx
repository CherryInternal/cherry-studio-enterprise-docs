# クイックスタート

このガイドでは、Cherry Studio エンタープライズ版の導入と設定を、購入から最初の AI メッセージ送信まで素早く完了できます。

## 流れの概要

```
ライセンス購入 → サービス導入 → 管理コンソール設定 → クライアント設定 → メッセージ送信
```

全体で **15〜30 分** 程度です。導入方法によって変わります。
