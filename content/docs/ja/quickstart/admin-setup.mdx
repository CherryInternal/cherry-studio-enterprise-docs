# 管理コンソール設定

導入完了後、管理コンソールで初期設定を行います。

## 管理コンソールにログイン

1. ブラウザで `http://your-server:3680` を開く
2. 既定の管理者アカウントでログイン：
   - ユーザー名：`admin`
   - パスワード：`admin123`（初回ログイン後、必ず変更）

## 必須設定ステップ

### 1. プロバイダーを設定

AI モデルのプロバイダーを追加して、ユーザーが AI を利用できるようにします：

1. **プロバイダー管理** に移動
2. **プロバイダー追加** をクリック
3. プロバイダー種別（OpenAI、Anthropic、Azure など）を選択
4. API Key と設定情報を入力
5. 接続テストを実行して保存

詳細ドキュメント： [プロバイダー管理](/ja/admin/service-provider)

### 2. グループを作成

グループはユーザーが利用できるモデルリソースを制御します：

1. **グループ管理** に移動
2. **グループ作成** をクリック
3. グループ名を設定（例：「開発」「マーケティング」）
4. グループで利用可能なモデルを選択

詳細ドキュメント： [グループ管理](/ja/admin/group-management)

### 3. ユーザーを追加

チームメンバーのアカウントを作成します：

1. **ユーザー管理** に移動
2. **ユーザー追加** をクリック
3. ユーザー情報（ユーザー名、メール、パスワード）を入力
4. 対応するグループに割り当て

詳細ドキュメント： [ユーザー管理](/ja/admin/user-management)

## 次のステップ

管理コンソールの設定が完了したら、[クライアント設定](/ja/quickstart/client-setup) に進んでください。
