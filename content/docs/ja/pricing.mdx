Cherry Studio は、個人開発者から大企業まで、幅広い規模のユーザーに向けて柔軟な料金プランを提供しています。

## バージョン選択ガイド

### Express 版 vs エンタープライズ版：主な違い

**機能は完全に同一**：Express 版とエンタープライズ版は機能面で完全に一致し、どちらもエンタープライズ機能をフル搭載しています。

**主要な違いは 2 点のみ**：

1. **プロバイダー設定方式**

   - Express 版：内蔵プロバイダーのみ利用可能。すぐに使えて設定不要
   - エンタープライズ版：完全自由に設定可能。カスタムプロバイダーや私有化モデルを追加可能

2. **価格体系**
   - Express 版：¥5,000 〜、人数制限なし
   - エンタープライズ版：¥50,000 〜（1-50 人）、人数に応じた階段価格

### どう選ぶべき？

- **Express 版を選ぶ**：迅速に導入したい、主要 LLM サービスを使いたい、カスタムプロバイダー不要
- **エンタープライズ版を選ぶ**：私有化モデルや自社サービス連携、特別なプロバイダー要件がある

## 各バージョンの詳細

### コミュニティ版（Community）

---

**対象ユーザー**：個人ユーザー、開発者
**導入方式**：単体利用
**技術サポート**：コミュニティサポート
**ライセンス**：AGPL 3.0 オープンソースライセンス

**主要機能**：

- 完全無料、利用期限なし
- 主要 AI モデルの基本的な対話機能
- ローカル導入でデータを完全に自己管理
- 内蔵 30+ プロバイダー（OpenAI、Anthropic、Google Gemini、国内 LLM など）
- **機能は豊富だが手動設定が必要**：各プロバイダーの API Key を自分で取得・設定

**プロバイダー利用方式**：

- プロバイダー一覧は提供されるが、アカウント登録はユーザー自身で実施
- API Key、Base URL などの設定を手動入力
- 技術的なユーザーや API キー管理に慣れた方向け

**利用シーン**：

- 個人の AI アシスタント
- 開発者の学習・試用
- 小規模チームの内部ツール
- コスト重視の個人ユーザー

### Express 版

---

**対象ユーザー**：スタートアップ、中小規模チーム
**導入方式**：プライベート導入
**技術サポート**：専用技術サポート（年額サービス費が必要）
**価格**：¥5,000 〜、人数制限なし

**主要機能**（エンタープライズ版と同等）：

- ✅ 基本機能：完全な AI 対話とマルチモデル対応
- ✅ プロバイダー管理：主要 LLM サービスを内蔵（カスタム不可）
- ✅ モデル管理：利用可能な AI モデルの管理・設定
- ✅ 従業員管理：企業ユーザーアカウントと組織構造
- ✅ 共有知識ベース：エンタープライズ向けベクトル知識ベースと文書管理
- ✅ 権限制御：役割ベースのアクセス制御（RBAC）
- ✅ データバックアップ：完全なバックアップ/復旧機構
- ✅ アシスタント管理：AI アシスタントのカスタム設定・管理
- ✅ エージェント管理：エンタープライズエージェントのワークフロー編成
- ✅ MCP 管理：MCP ツールの統合・管理
- ✅ ミニプログラム管理：企業向けミニプログラムのエコシステム

**プロバイダー利用方式**：

- **内蔵プロバイダーを利用**：主要 LLM サービスが事前設定済みで **すぐに使用可能**
- **カスタム不可**：独自の API Key や他プロバイダーの追加は不可
- 複雑な API Key 設定は不要で、素早く導入したい企業に最適

**プロバイダー制限**：

- システム内蔵のプロバイダーとモデルのみ利用可能
- 私有化導入のモデルサービスは追加不可
- 自身の API Key は利用不可

**利用シーン**：

- 迅速な導入と即時利用を希望する中小企業
- カスタムプロバイダー設定が不要なチーム
- 機能は必要だがプロバイダー要件が少ない企業
- 人数制限なく利用したいチーム

**導入方式**：プライベート導入
**技術サポート**：専用技術サポート（年額サービス費が必要）
**価格**：¥50,000 〜（1-50 人）、人数に応じた階段価格

**主要機能**（Express 版と同等）：

- ✅ プロバイダー管理：完全自由設定（カスタムプロバイダー対応）
- ✅ モデル管理：利用可能な AI モデルの管理・設定
- ✅ 従業員管理：企業ユーザーアカウントと組織構造
- ✅ 共有知識ベース：エンタープライズ向けベクトル知識ベースと文書管理

- ✅ エージェント管理：エンタープライズエージェントのワークフロー編成

**プロバイダー利用方式**：

- 企業内ネットワークでのモデルサービス接続に対応
- 柔軟なプロバイダー権限の割り当てと管理

**利用シーン**：
