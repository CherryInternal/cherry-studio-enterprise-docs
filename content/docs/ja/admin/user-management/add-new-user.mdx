## 操作手順 <a href="#heading8ca0746f50e147119c6c47d6dded5dd2-cao-zuo-liu-cheng-0" id="heading8ca0746f50e147119c6c47d6dded5dd2-cao-zuo-liu-cheng-0"></a>

1. ユーザー管理画面の右上で **+ ユーザー追加** をクリック。
2. スライド/モーダルで以下を設定します：

<img src="/assets/images/ja/admin/user-management-add-user-modal_ja.webp" alt="新規ユーザー追加モーダル" />
## パラメータの詳細 <a href="#heading8ca0746f50e147119c6c47d6dded5dd2-pei-zhi-can-shu-xiang-jie-0" id="heading8ca0746f50e147119c6c47d6dded5dd2-pei-zhi-can-shu-xiang-jie-0"></a>

<table><thead><tr><th width="105.7265625">項目</th><th width="76.95703125">必須</th><th width="139.546875">形式</th><th>説明</th></tr></thead><tbody><tr><td><strong>所属グループ</strong></td><td><strong>はい</strong></td><td>ドロップダウン</td><td><strong>最重要設定</strong>。利用可能なモデル（Model Providers）とアシスタントが決まります。</td></tr><tr><td><strong>ユーザー名</strong></td><td><strong>はい</strong></td><td>3 文字以上</td><td>氏名や社員番号を推奨。システム内で一意である必要があります。</td></tr><tr><td><strong>パスワード</strong></td><td><strong>はい</strong></td><td>6 文字以上</td><td>初回ログイン用パスワード。</td></tr><tr><td><strong>グループ管理者</strong></td><td>いいえ</td><td>トグル（既定 OFF）</td><td>ON にすると、所属グループの管理者として設定されます。</td></tr></tbody></table>

3. **ユーザー作成** をクリックして完了します。

<Callout type="info">
**Tip:** セキュリティのため、初回ログイン後に初期パスワードを変更するよう案内してください。
</Callout>
