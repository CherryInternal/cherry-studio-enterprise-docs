大量のユーザーアカウントを一度に作成する必要がある場合、手動での個別追加は非効率です。一括インポート機能は **Excel (.xlsx)** または **CSV (.csv)** ファイルで複数ユーザーを一度にインポートでき、存在しないグループも自動作成されます。

<Callout type="info">
**入口：** ユーザー管理ページまたはグループ管理ページの右上にある**インポート**ボタン。
</Callout>

## 操作フロー

インポートは 3 ステップで完了します：

### ステップ 1：ファイルアップロード

<img src="/assets/images/ja/admin/user-management-import-step1_ja.webp" alt="一括インポート - ファイルアップロード" />

1. **Excel テンプレートをダウンロード**または **CSV テンプレートをダウンロード**をクリックして標準テンプレートを取得。
2. テンプレートの形式に従ってユーザーデータを入力。
3. 完成したファイルをアップロードエリアにドラッグ＆ドロップ、またはクリックして選択。
4. **次へ**をクリックしてデータプレビューに進む。

### ステップ 2：データプレビュー

ファイル解析後、プレビュー表が表示されデータを確認できます。フォーマットエラーや必須項目の不足はハイライト表示されます。

### ステップ 3：インポート実行

確認後、実行をクリックしてユーザーとグループを一括作成。完了後、結果サマリー（成功数、スキップ数、失敗数）が表示されます。

<Callout type="warn">
**注意事項：**
* テンプレートのヘッダー行は変更しないでください。
* 既存のユーザー名はスキップまたは更新を選択可能。
* 存在しないグループは自動作成されます（管理者権限が必要）。
* 一括インポート前に、少量のデータでテストすることを推奨します。
</Callout>
