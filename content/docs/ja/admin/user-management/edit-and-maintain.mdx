# 3. 編集と管理

#### ユーザー情報の編集 <a href="#heading8ca0746f50e147119c6c47d6dded5dd2-bian-ji-yong-hu-xin-xi-0" id="heading8ca0746f50e147119c6c47d6dded5dd2-bian-ji-yong-hu-xin-xi-0"></a>

![ユーザー編集モーダル](/assets/images/zh/admin/user-management-edit-user-modal_zh.png)

グループ権限の変更やパスワードのリセットを行う場合：

1. ユーザー一覧で対象ユーザーを探します。
2. **操作** 列の **⋮**（その他）をクリック。
3. **編集** を選択。
4. **よくある操作**：
   * **パスワードリセット**："新しいパスワード" に入力。
   * **権限変更**：**所属グループ** を変更すると即時反映。
   * **アカウント停止**：**アクティブ状態** をオフにすると即座にログイン不可。
5. **ユーザー更新** をクリックして保存。

#### ユーザー削除 <a href="#heading8ca0746f50e147119c6c47d6dded5dd2-shan-chu-yong-hu-0" id="heading8ca0746f50e147119c6c47d6dded5dd2-shan-chu-yong-hu-0"></a>

退職者や不要なテストアカウントの場合：

1. **操作** 列の **⋮** をクリック。
2. **削除**（赤色）を選択。

![](/assets/images/zh/admin/user-management-edit-user-1_zh.png)

3. 確認ダイアログで削除を確定します。

![](/assets/images/zh/admin/user-management-edit-user-2_zh.png)

<Callout type="error">
**Warning:** 削除は**取り消し不可**です。削除後、個人の知識ベース設定や会話履歴は復旧できない場合があります。
</Callout>
