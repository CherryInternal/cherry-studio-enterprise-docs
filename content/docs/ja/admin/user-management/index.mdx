## 主な機能

このモジュールでは以下の操作が可能です：

* **ライフサイクル管理**：アカウント作成から情報変更（メール、パスワード、電話）、無効化/削除まで対応。
* **きめ細かな権限付与**：**グループ**を使ってユーザーをモデルリソースに紐付け、「開発は Claude、マーケは GPT-4」などの差別化が可能。
* **セキュリティ遮断**：**アクティブ/非アクティブ**の切り替えで、退職や異常時に即座にログイン/API を遮断。
* **API ゲートウェイ認証**：開発者や自動化スクリプト向けに **API Key** を発行し、クライアントなしでモデルを呼び出し可能。
* **多面的な監査**：ユーザー名検索、状態/グループのフィルタリングで対象ユーザーを迅速に特定。

## クイックスタート

管理コンソール左側のナビゲーションから **ユーザー管理** を選択してください。

<Callout type="warn">
**重要:** ユーザー作成後の**権限範囲**は所属する **グループ** に依存します。ユーザー追加前に **グループ管理** で組織構造とモデル権限戦略を設計してください。
</Callout>
