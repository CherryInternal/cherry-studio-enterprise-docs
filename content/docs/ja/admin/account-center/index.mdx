## 入口

管理コンソール**左下の人物アイコン**をクリックし、ポップアップメニューから**アカウント設定**を選択すると、アカウントセンターに入れます。

<img src="/assets/images/ja/admin/account-center-entry_ja.webp" alt="アカウントセンター入口" />

<Callout type="info">
アカウントセンターは**ユーザーセルフサービス**の入口で、ログイン中のすべてのアカウントから利用できます。管理者向けサイドバーの「注文管理」「ユーザー管理」などとは役割が異なります。
</Callout>

## 機能概要

アカウントセンターには 3 つのタブがあります：

| タブ | 説明 |
| --- | --- |
| **アカウント設定** | 個人情報（ユーザー名・メール）の確認とログインパスワードの変更を行います。 |
| **マイ注文** | 現在のアカウントに紐づくすべての注文履歴を確認します。 |
| **関連組織** | 参加または作成したすべての組織を一覧表示し、対応する管理コンソールへ遷移したり、招待コードで新しい組織に参加したりできます。 |

ページ上部には**クライアントをダウンロード**入口もあり、Cherry Studio クライアントをすぐにダウンロードして AI サービスを利用開始できます。
