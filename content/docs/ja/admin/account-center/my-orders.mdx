アカウントセンターで **マイ注文** タブをクリックすると、現在のアカウントに紐づくすべての注文履歴を確認できます。

<img src="/assets/images/ja/admin/account-center-orders_ja.webp" alt="マイ注文" />

## 一覧の項目説明

| 項目 | 説明 |
| --- | --- |
| **商品名** | 購入した商品（例：「Cherry Studio Enterprise Express 版」「Token チャージ」など）。 |
| **注文番号** | システムが自動生成する一意の注文 ID。 |
| **関連組織** | 注文が紐づく組織。 |
| **作成日時** | 注文が作成された日付。 |
| **金額** | 注文金額。 |
| **ステータス** | 注文の現在の状態（完了 / 支払い待ち / クローズ など）。 |

## 検索とフィルター

- **検索ボックス**：注文番号、商品名、組織名でクイック検索できます。
- **タイプフィルター**：商品タイプ（全タイプ、プラン、Token チャージ など）で絞り込みます。
- **ステータスフィルター**：注文ステータスで絞り込みます。

<Callout type="info">
管理者向けの「注文管理」（サイドバー入口）では、組織全体のすべての注文を閲覧・管理できます。アカウントセンターの「マイ注文」は、現在ログインしているアカウントに関連する注文のみを表示します。
</Callout>
