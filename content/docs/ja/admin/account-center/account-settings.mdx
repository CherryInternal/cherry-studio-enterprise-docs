アカウントセンターで **アカウント設定** タブをクリックすると、個人情報とログインパスワードを管理できます。

<img src="/assets/images/ja/admin/account-center-settings_ja.webp" alt="アカウント設定" />

## 個人情報

アカウントの基本情報は読み取り専用で表示されます：

| 項目 | 説明 |
| --- | --- |
| **ユーザー名** | 登録時に使用したユーザー名です。 |
| **メールアドレス** | アカウントに紐づくメールアドレスです。 |

<Callout type="info">
ユーザー名やメールアドレスを変更したい場合は、組織管理者にお問い合わせいただくか、サポートチケットからご申請ください。
</Callout>

## パスワード変更

以下の手順でログインパスワードを変更します：

1. **現在のパスワード** 欄に旧パスワードを入力し、本人確認を行います。
2. **新しいパスワード** 欄に新しいパスワード（6 文字以上）を設定します。
3. **新しいパスワードの確認** 欄に同じ新しいパスワードを再入力します。
4. **保存** をクリックします。

変更が完了したら、次回のログインから新しいパスワードをご利用ください。

<Callout type="warn">
アカウントのセキュリティ確保のため、定期的にパスワードを変更し、他のサービスとの使い回しは避けてください。
</Callout>
