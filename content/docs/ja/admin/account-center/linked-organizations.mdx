アカウントセンターで **関連組織** タブをクリックすると、参加または作成したすべての組織を一覧表示できます。組織を切り替えたり、新しい組織に参加したりすることも可能です。

<img src="/assets/images/ja/admin/account-center-organizations_ja.webp" alt="関連組織" />

## 組織リスト

現在のアカウントに紐づくすべての組織がカード形式で表示されます。各カードには以下が表示されます：

- **組織名** と **ステータス**（例：「アクティブ」）
- **役割**（オーナー / 管理者 / メンバー）
- **管理コンソールに入る** ボタン — クリックすると該当組織の管理コンソールに直接遷移します

<Callout type="info">
管理コンソールに入れるのはオーナーと管理者のみです。一般メンバーのカードでは「管理コンソールに入る」ボタンが利用できない場合があります。
</Callout>

## 企業招待コードで組織に参加

組織管理者から招待リンクや企業招待コードを受け取った場合：

1. 入力欄に企業招待コードまたは招待リンクを貼り付けます。
2. **参加** ボタンをクリックします。
3. 参加が成功すると、新しい組織が上記の組織リストに表示されます。

<Callout type="info">
管理者がメンバーを招待する詳細手順については <DynamicLink href="/[lang]/docs/admin/user-management/invite-members">メンバー招待</DynamicLink> を参照してください。
</Callout>

## 新しい組織を作成

リスト下部の **+ 組織を作成** ボタンをクリックすると、新しい組織を作成できます。作成後はその組織のオーナーとなり、企業認証と支払い開通のフローを完了する必要があります。

<Callout type="info">
組織作成の完全なフロー（登録 → 組織を作成 → プラン選択 → 企業認証 → 支払い）は <DynamicLink href="/[lang]/docs/quickstart">クイックスタート</DynamicLink> を参照してください。
</Callout>
