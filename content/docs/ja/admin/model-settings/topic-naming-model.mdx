---
title: "トピック命名モデル"
---

import { Callout } from 'fumadocs-ui/components/callout'

**定義**：ユーザーが新規会話で最初のメッセージを送信すると、バックグラウンドでこのモデルが要約し短いタイトルを生成します（例：「金曜会議の議事録を作成して…」→「金曜会議の議事録」）。

<Callout type="warn">
**⚠️ コスト/性能注意** ここに **推論モデル**（`DeepSeek-R1`, `OpenAI o1` など）を設定するのは避けてください。

* **理由**：推論モデルは回答前に長い思考を行うため、簡単なタイトル生成には過剰です。**非常に遅く**、不必要に高コストになります。
</Callout>

* **ベストプラクティス**：
  * **推奨モデル**：`Qwen 2.5 7B`、`GPT-5.1-mini`、`Gemini Flash`。
  * **重要要件**：**高速・指示追従・安価**。"タイトルに要約"を理解できれば十分です。
