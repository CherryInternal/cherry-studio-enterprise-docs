# 既定モデル設定

**パス：** 管理コンソールサイドバー > **既定モデル**

このページでは、Cherry Studio Enterprise の各業務シーンで使用する**グローバルなフォールバックモデル**を定義します。適切な設定により、体験（速度/品質）とトークンコストのバランスを取れます。

<img src="/assets/images/zh/admin/default-model-settings-overview_zh.webp" alt="" />
<Callout type="info">
**前提条件:** プルダウンに表示されるのは **[モデル管理]** で追加・有効化されたモデルのみです。見つからない場合は先にモデル管理で追加してください。
</Callout>
