## 機能概要

グループの一括インポートでは、管理者が Excel または CSV ファイルをアップロードして複数のグループを一度に作成できます。組織構造を素早く構築する場合に最適です。

## 操作手順

1. 管理コンソール → グループ管理に移動
2. ページ右上の **インポート** ボタンをクリック
3. ダイアログから標準テンプレート（Excel または CSV 形式）をダウンロード
4. テンプレートのルールに従ってグループデータを入力
5. 入力済みファイルをアップロードし、**次のステップ** をクリックしてインポートを開始

<img src="/assets/images/ja/admin/group-management-batch-import_ja.webp" alt="グループの一括インポート" />

### テンプレートフィールド説明

| フィールド名 | 必須 | 説明 |
| --- | --- | --- |
| **name** | はい | グループ名、一意の識別子 |
| **description** | いいえ | グループ説明 |

## 注意事項

- 最初の行はヘッダーです。変更しないでください
- 既存のグループ名はスキップまたは更新できます
