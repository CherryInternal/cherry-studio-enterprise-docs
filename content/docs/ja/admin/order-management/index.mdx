<Callout type="info">
**注文管理** は **Express 版** のみで利用可能です。Premium 版はライセンスベースの認証を使用し、注文フローはありません。
</Callout>

**注文管理**では、プラン購入や Token チャージを含む、組織のすべての注文記録を表示・管理できます。
