管理コンソール左側のナビゲーションから「Orders」をクリックすると、すべての注文記録を確認できます。

<img src="/assets/images/ja/admin/order-management-list_ja.webp" alt="注文管理一覧" />

## カラム説明

| カラム | 説明 |
| --- | --- |
| **Order No.** | システムが自動生成した一意の注文番号。 |
| **Product** | 購入した商品名（例：「Token 充値」「Cherry Studio Enterprise 極速版」）。 |
| **Amount** | 注文金額。 |
| **Status** | 注文の現在のステータス（下記参照）。 |
| **Payment Method** | 銀行振込または Alipay。 |
| **Created At** | 注文作成日時。 |

## 検索とフィルタ

- **検索バー**：注文番号または商品名で検索できます。
- **ステータスフィルタ**：「All」「Completed」「Closed」などのステータスで絞り込みが可能です。

## 注文ステータス

| ステータス | 説明 |
| --- | --- |
| **待支付** | 注文作成済み、支払い待ち。 |
| **已支付** | 支払い受領済み、システム確認およびサービス開通待ち。 |
| **已完成** | 支払い確認完了 — サービスが開通済み、または Token がチャージ済み。 |
| **已关闭** | 注文がクローズ（支払いタイムアウトまたは手動キャンセル）。 |

<Callout type="info">
**待支付**の注文には残り支払い時間のカウントダウンが表示されます。「支付」をクリックして支払いを続行するか、「取消」をクリックして注文をクローズできます。
</Callout>
