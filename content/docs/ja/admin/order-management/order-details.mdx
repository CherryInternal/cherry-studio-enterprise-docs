---

注文をクリックすると、注文の詳細を確認できます。

<img src="/assets/images/ja/admin/order-management-detail_ja.webp" alt="注文詳細" />

## 注文情報

| フィールド | 説明 |
| --- | --- |
| **Order No.** | 一意の注文番号。 |
| **Company** | 注文企業名。 |
| **Created At** | 注文作成日時。 |
| **Paid At** | 支払い完了日時。 |
| **Payment Method** | 銀行振込または Alipay。 |

## 商品情報

| フィールド | 説明 |
| --- | --- |
| **Product Name** | 購入した商品（例：「Token 充値」）。 |
| **Product Code** | 商品のシステムコード（例：`token-recharge`）。 |
