**パス：** **About** > **アプリ情報**

このパネルは現在の稼働状況とダウンロード入口を提供し、システム健康状態の一次ダッシュボードです。

**注意:** **サービス状態** が異常でもフロントは表示される場合がありますが、モデル呼び出しやファイル解析などのバックエンド機能は失敗します。

#### リソースダウンロード

クライアント/サーバーのバージョン整合のため、以下を利用できます：

* **クライアントダウンロード:** Windows / macOS クライアント。
* **ドキュメント:** 公式マニュアル（本ドキュメント）。
* **Docker イメージ:** Docker Hub へのリンク。
