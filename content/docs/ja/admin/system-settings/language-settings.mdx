title: "言語設定"
description: "管理パネルの表示言語を設定します。自動検出・簡体字中国語・英語・日本語に対応。"
---

## 設定方法

1. **システム設定** > **言語** を開く。
2. **言語を選択** ドロップダウンから希望する言語を選択。

<img src="/assets/images/ja/admin/system-settings-language_ja.webp" alt="言語設定" />

## 選択可能な言語

| オプション | 説明 |
|---|---|
| **自動** | ブラウザの言語設定に基づいて自動的に判定。 |
| **简体中文** | 管理パネルを簡体字中国語で表示。 |
| **English** | 管理パネルを英語で表示。 |
| **日本語** | 管理パネルを日本語で表示。 |

言語設定は現在のブラウザにのみ適用されます。変更は即座に反映され、ページの再読み込みは不要です。
