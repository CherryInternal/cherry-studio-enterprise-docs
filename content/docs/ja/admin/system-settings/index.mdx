# システム設定

正しい設定は以下の体験に不可欠です：

* **モデル接続性**：制限されたネットワーク下で OpenAI/Anthropic に接続。
* **知識ベース精度**：OCR で PDF 解析品質を向上。
* **リアルタイム連携**：ニュース検索などのオンライン能力。
* **性能最適化**：同期頻度とローカルストレージ管理。
