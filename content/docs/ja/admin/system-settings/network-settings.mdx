## 設定手順

1. **システム設定** > **ネットワーク設定** へ移動。
2. 企業のプロキシサーバー/ゲートウェイが正常に稼働していることを確認（社内から統一出口経由で外部 API にアクセスする場合など）。
3. 下記の入力欄にプロキシアドレスを入力。

## プロキシポート参考

プロキシソフト/ゲートウェイにより監視ポートは異なります。実際にお使いのプロキシサービスの設定画面にある「ローカル監視ポート」を確認し、対応する `http://<アドレス>:<ポート>` を入力してください。

<Callout type="warn">
**注意**：

1. `socks5://` は使用しないでください。Cherry Studio は主に `http://` プロキシに対応します。
2. 設定後、**モデル管理** の **Check** で接続確認を行ってください。
</Callout>
