## 設定手順

1. **システム設定** > **ネットワーク設定** へ移動。
2. ローカルのプロキシソフト（Clash, v2rayN, Surge など）を起動。
3. 下記の入力欄にプロキシアドレスを入力。

## よくあるプロキシポート

プロキシソフトにより既定ポートは異なります。ローカルの監視ポートを確認してください：

* **Clash / CFW**: `7890`
* **v2rayN**: `10809`
* **Surge**: `6152`

<Callout type="warn">
**注意**：

1. `socks5://` は使用しないでください。Cherry Studio は主に `http://` プロキシに対応します。
2. 設定後、**モデル管理** の **Check** で接続確認を行ってください。
</Callout>
