# 3. 検索エンジン設定

#### 対応プロバイダー

Cherry Studio は複数の検索 API を統合しています。用途に応じて選択してください：

* **Tavily**（推奨）：AI 向けに最適化され、ノイズが少なく本文抽出が可能。
* **Zhipu**：中国語環境に適した検索。
* **SearxNG**：プライバシー重視の自前運用向け。
* **ExaSearch / Bocha**：その他のニューラル検索。

#### 設定手順

1. **既定の検索プロバイダー** で API Key を持つプロバイダーを選択（例：`Tavily`）。
2. 下の設定欄に認証情報を入力。

<img src="/assets/images/ja/admin/system-settings-search-engine_ja.webp" alt="" />
#### パラメータ詳細

| パラメータ | 説明 | 入手/設定方法 |
| ------------- | ------------------------ | --------------------------------------- |
| **既定の検索プロバイダー** | **必須**。実際に使用する検索サービス。 | ドロップダウンで選択（例：`Tavily`）。 |
| **API Host** | プロバイダーのエンドポイント。 | 通常は既定値。自前 SearxNG は自分のサーバーを入力。 |
| **API Key** | 個人のアクセスキー。 | 各プロバイダーのコンソールで取得。 |
| **ユーザー名** | `SearxNG` のみ必要。 | ベーシック認証がある場合に入力。 |

#### 検証と確認

設定後、**Web 検索** を有効にしたアシスタントを作成し、「今日の AI 最新ニュースは？」と質問します。リンク引用が返れば成功です。
