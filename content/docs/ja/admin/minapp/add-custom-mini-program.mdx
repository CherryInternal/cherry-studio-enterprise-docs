---

import { Callout } from 'fumadocs-ui/components/callout'

管理者は標準に準拠した Web URL をミニプログラム化できます。

## 手順

1. 右上の **+ ミニプログラム追加** をクリック。

<img src="/assets/images/ja/admin/mini-program-add-custom_ja.webp" alt="" />
2. ダイアログで設定します：

<table><thead><tr><th width="91.7421875">項目</th><th width="94.125">フィールド</th><th>説明</th></tr></thead><tbody><tr><td><strong>視覚識別</strong></td><td><strong>ロゴアップロード</strong></td><td><strong>PNG</strong> / <strong>WebP</strong> 推奨。透過背景が望ましい。推奨サイズ <code>200x200px</code>、最大 <code>200KB</code>。</td></tr><tr><td></td><td><strong>名称</strong></td><td>アイコン下に表示される文字（短め推奨）。</td></tr><tr><td><strong>コア設定</strong></td><td><strong>URL</strong></td><td><strong>プロトコル必須</strong>（例：<code>https://</code>）。<br />例：<code>https://docs.cherry-ai.com</code></td></tr><tr><td><strong>スタイル調整</strong></td><td><strong>枠線</strong></td><td>ロゴ色と背景が近い場合に枠線を追加。</td></tr><tr><td></td><td><strong>背景色</strong></td><td>ロゴ背景色を設定（Hex 例：<code>#FFFFFF</code>）。透過 PNG の場合は識別性が向上します。</td></tr></tbody></table>

3. **ミニプログラム追加** をクリックして保存。

<Callout type="warn">
**URL の制限:** `https://` から始まる安全な URL を使用してください。クライアントの安全ポリシーにより `http://` はブロックされる場合があります。
</Callout>
