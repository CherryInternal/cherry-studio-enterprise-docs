# ミニプログラム管理

<Callout type="info">
**ナビゲーション：** サイドバー → リソースライブラリ → ミニプログラム
</Callout>

**ミニプログラム管理** は Cherry Studio Enterprise の「ワンストップ AI ワークスペース」を実現する中核機能です。外部 AI ツール、社内システム、Web サービスをクライアントのサイドバーに統合し、ブラウザ切替なしで利用できます。

DeepSeek、Kimi などの主要 AI Web サービスの統合に加え、自社ドキュメントライブラリや LLM の Web UI も埋め込めます。
