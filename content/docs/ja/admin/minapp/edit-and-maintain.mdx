# 3. 編集と管理

**カスタムミニプログラム** はいつでも設定変更できます。

<img src="/assets/images/ja/admin/mini-program-edit-1_ja.webp" alt="" />
#### 情報の更新

1. 一覧で対象（タイプ：カスタム）を見つける。
2. 操作欄の **編集** をクリック。
3. **URL** の更新（移行など）や **ロゴ** の差し替えを行う。
4. **ミニプログラム更新** をクリックして反映。

#### ミニプログラムの削除

<img src="/assets/images/ja/admin/mini-program-edit-2_ja.webp" alt="" />
* 赤い **削除** ボタンで永久削除。
* **注意:** 削除は即時反映され、全社員のクライアントから消えます。
