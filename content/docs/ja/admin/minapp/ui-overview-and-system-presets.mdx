# 1. 画面概要とシステムプリセット

**ミニプログラム管理** ページでは、企業環境で利用可能なアプリ一覧を確認できます。

#### 画面構成

![](/assets/images/zh/admin/mini-program-overview_zh.webp)

一覧は 2 種類に分かれます：

1. **システムプリセット:** Cherry Studio が厳選した主要 AI サービス（DeepSeek、ChatGPT、通義千問 など）。URL は公式管理、管理者は有効/無効のみ。
2. **カスタム:** 管理者が追加した社内/外部リンク。

<Callout type="info">
**なぜ削除できないアプリがある？** システムプリセットは安定性のためにハードコードされており、**無効化** はできても削除はできません。
</Callout>

#### クイック有効化/無効化

右側の **有効** 列のトグルで、クライアントのサイドバー表示を切り替えます。

* **ON:** クライアント再起動後、全社員に表示。
* **OFF:** 非表示にするが設定は保持。
