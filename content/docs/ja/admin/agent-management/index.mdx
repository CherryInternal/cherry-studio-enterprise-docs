# エージェント管理

**エージェント (Agent)** は Cherry Studio Enterprise の開発シナリオ向け高度な AI 機能です。プロンプトと知識ベースを活用した対話型の「アシスタント」とは異なり、エージェントは**マルチモデル連携**（メインモデル/プランニングモデル/高速モデル）、**権限モード制御**、**スキルバインド**、**MCP ツール統合**を備え、複雑なマルチステップタスクを自律的に遂行できます。

このモジュールで、管理者はチーム向けの専門 AI エージェントを作成・設定し、グループごとにユーザーへ配布できます。

<Callout type="info">
**コア価値：**

* **マルチモデル連携：** タスクフェーズごとに最適なモデルを設定 — メインモデルで推論、プランニングモデルで分析、高速モデルで軽量処理。
* **権限ガバナンス：** デフォルト権限モードと最大権限モードを管理者が制御し、企業セキュリティポリシーを遵守。
* **スキル拡張：** リソースライブラリからスキルをバインドし、コード分析やファイル処理などの専門能力を付与。
* **MCP ツール統合：** MCP サーバーを接続し、外部ツールやデータソースを呼び出し可能。
</Callout>

<img src="/assets/images/ja/admin/agent-management-list_ja.webp" alt="エージェント管理一覧" />

エージェント一覧には作成済みの全エージェントが表示され、名前検索やグループフィルタリングが可能です。各エージェントの名前、所属グループ、モデル、説明が確認できます。
