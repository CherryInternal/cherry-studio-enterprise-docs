# エージェントスキル

**エージェントスキル**はリソースライブラリ内のエージェント専用の再利用可能な機能モジュールです。管理者は**手動アップロード**または**マーケットプレイスからダウンロード**の 2 つの方法でスキルを追加できます。追加されたスキルはエージェントの作成・編集時にバインドして使用できます。

<Callout type="info">
**ナビゲーション：** サイドバー → リソースライブラリ → エージェントスキル（最初のタブ）
</Callout>

<img src="/assets/images/ja/admin/agent-skills-list_ja.webp" alt="エージェントスキル一覧" />

## スキル一覧

スキル一覧には追加済みの全エージェントスキルが以下の情報とともに表示されます：

<table><thead><tr><th width="100">フィールド</th><th>説明</th></tr></thead><tbody><tr><td><strong>名前</strong></td><td>スキルの識別名。</td></tr><tr><td><strong>グループ</strong></td><td>スキルが属するグループ。どのユーザーグループがアクセスできるかを決定します。</td></tr><tr><td><strong>説明</strong></td><td>スキルの機能説明。</td></tr><tr><td><strong>操作</strong></td><td>編集、詳細表示、削除などの操作。</td></tr></tbody></table>

検索ボックスで名前からスキルをすばやく検索できます。

## スキルの追加

右上の**追加**ボタンをクリックします。2 つの方法が利用可能です：

<img src="/assets/images/ja/admin/agent-skills-add-menu_ja.webp" alt="スキル追加メニュー" />

### 手動アップロード

企業独自のスキルやサードパーティのカスタムスキル向け。スキルファイル（ZIP または Markdown 形式）をアップロードし、名前や説明などのメタデータを入力します。

### マーケットプレイスからダウンロード

スキルマーケットプレイスブラウザを開き、Cherry Studio 公式マーケットプレイスから検証済みのスキルを選択・ダウンロードできます。

<Callout type="info">
**ワークフロー：** リソースライブラリでスキルを追加 → エージェントの「スキル」タブでスキルを有効化 → エージェントが実行時にそのスキルの能力を使用可能。
</Callout>
