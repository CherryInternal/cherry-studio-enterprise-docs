**知識ベース (Knowledge Base)** は Cherry Studio Enterprise の **RAG (Retrieval-Augmented Generation)** の中核です。大規模モデルの「外部脳」として、社内の文書・Web・メモなどをベクトル化して保存します。

ユーザーがアシスタントと対話する際、システムはまず知識ベースから関連情報を検索してモデルに渡し、社内事実に基づく回答を実現します。これにより幻覚を減らし、データ安全性を高めます。
