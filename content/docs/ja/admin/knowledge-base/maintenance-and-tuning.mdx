# 3. 保守とチューニング

#### 設定の変更

知識ベース一覧で **設定** をクリックし、名称・説明・リランキング戦略を変更できます。

![](/assets/images/zh/admin/knowledge-base-maintenance-tuning_zh.webp)

<Callout type="info">
**チャンクサイズを変更するタイミング**：回答が途中で切れている場合は **チャンクサイズ** が小さすぎる可能性があります。無関係な文が多い場合は大きすぎる可能性があります。変更には再アップロード/再処理が必要になることが多いです。
</Callout>

#### 検索品質の検証

正しく設定できているか、以下で確認してください：

1. **アシスタント管理** でテストアシスタントを作成。
2. **知識ベース設定** で作成した知識ベースを関連付け。
3. プレビューまたはクライアントで、その知識ベースにしかない質問を行う（例：「遅刻控除の規定は？」）。
4. **成功の目安:** 正しく回答され、回答にホバーすると **引用元** が文書を指す。
