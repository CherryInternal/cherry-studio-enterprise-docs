# 6. API 呼び出しログ

最下層の監査データで、各 API 呼び出しの完全な記録を保持します。

![](/assets/images/zh/admin/usage-analytics-call-logs-1_zh.png)

#### ログ詳細

右側の **表示（目のアイコン）** をクリックすると詳細カードが表示されます：

![](/assets/images/zh/admin/usage-analytics-call-logs-2_zh.png)

1. **ユーザー概要:** 呼び出し元と日時（秒単位）。
2. **モデル情報:** モデル ID（例：`google/gemini-pro`）とプロバイダー。
3. **トークン監査:**
   * **入力 Token:** 消費量。
   * **出力 Token:** 消費量。
   * **総 Token:** 今回の請求対象トークン総量。

#### 利用シーン

* **障害調査:** 返信切れやエラーの報告時、タイムスタンプで該当ログを探し、トークン超過を確認。
* **コンプライアンス監査:** 特定期間の呼び出しが規定に沿っているか確認。
