## 指標の説明

* **総リクエスト数:** プロバイダーへの API 呼び出し成功回数。
* **総 Token:** 入力+出力トークンの合計。コスト算定の最重要指標。
* **入力 Token:** AI に送信されたプロンプトと文脈長。
* **出力 Token:** AI の返信長。
* **アクティブユーザー:** 選択期間内に呼び出しを行ったユニークユーザー数。
* **アクティブモデル:** 呼び出されたモデル数。
* **中断リクエスト:** ユーザー停止/ネットワーク/タイムアウトによる失敗。
* **平均 Token/リクエスト:** 会話の平均複雑度。高いほど複雑/長文。

<Callout type="warn">
**運用ヒント:** **中断リクエスト** が増えたら **システム設定 > Model Provider** の接続状態や外部ネットワークを確認してください。
</Callout>
