## 操作方法

右上の **ユーザー別 / グループ別** をクリックして切り替えます。

* **グループ別:** 部門ごとのリソース消費を可視化。例：開発部は総務より消費が多い。
* **ユーザー別:** いわゆるヘビーユーザーを特定。青は入力、緑は出力で、長文読解型か生成型かを判断できます。
