# 3. トレンドと分布の可視化

#### 1. 利用トレンド

<img src="/assets/images/zh/admin/usage-analytics-trend-distribution-1_zh.webp" alt="" />
{/* TODO: Add additional screenshots, see MISSING_IMAGES.md */}

折れ線グラフでトークン消費の時系列を表示します。

* **青線:** 総トークン消費。
* **緑線:** 参照指標（通常は出力トークン）。
* **価値:** ピーク時間を把握し、同時接続枠を計画。

#### 2. プロバイダー分布

<img src="/assets/images/zh/admin/usage-analytics-trend-distribution-2_zh.webp" alt="" />
{/* TODO: Add additional screenshots, see MISSING_IMAGES.md */}

円グラフでプロバイダー別のトラフィック比率を表示します。

* **用途:** 有料/無料・自社モデルの比率を監視しコスト最適化。

#### 3. モデル分布

<img src="/assets/images/zh/admin/usage-analytics-trend-distribution-3_zh.webp" alt="" />
{/* TODO: Add additional screenshots, see MISSING_IMAGES.md */}

棒グラフで消費量上位モデルを表示します。

* **操作:** ホバーで詳細数値を表示。
* **価値:** 利用傾向を把握。`gpt-4` が高い場合、非重要タスクを `gpt-4o-mini` や `deepseek-chat` に誘導。
