# 1. フィルタとコントロール

ページ上部で詳細な条件を指定し、分析範囲を定義できます。下部のグラフ/一覧はリアルタイムで連動します。

<img src="/assets/images/ja/admin/usage-analytics-filters-controls_ja.webp" alt="" />
### 機能説明

| UI 要素 | 説明 | ビジネスシーン |
| -------- | ----------------------- | ----------------------- |
| **全グループ** | 組織構造（例：製品部、開発部）で絞り込み | 部門別の AI 投資対効果を評価 |
| **全ユーザー** | 特定ユーザーを検索・選択 | 異常な高頻度利用やトークン消費の監査 |
| **期間** | 今日/直近 7 日/30 日、またはカスタム | 導入前後の利用変化を比較 |
| **更新** | データ再読み込み | 最新の監視データを取得 |

<Callout type="info">
**注意:** 分散ログ集計の都合で 1〜2 分の遅延が発生する場合があります。
</Callout>
