ランキングは高頻度利用者を素早く把握するリストです。

<img src="/assets/images/ja/admin/usage-analytics-leaderboard_ja.webp" alt="" />
## 項目

* **順位:** 消費量の降順。
* **名前:** ユーザー表示名。
* **グループ:** 所属部門。
* **リクエスト数:** 活動頻度。
* **トークン使用量:** 合計消費量。

<Callout type="info">
**ベストプラクティス:** Top 5 を定期的に確認。正常なら十分なクォータを確保し、急増の場合は API Key の不正利用を防ぐために確認してください。
</Callout>
