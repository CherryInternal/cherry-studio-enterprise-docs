# 1. ライセンス状態の概要

**ライセンス管理** ページの上部パネルは、システム健康状態の「バロメーター」です。

![ライセンス取り込み](/assets/images/zh/admin/authorization-import_zh.png)

![検証失敗](/assets/images/zh/admin/authorization-validation-failed_zh.png)

![ライセンス有効・正常稼働](/assets/images/zh/admin/authorization-valid_zh.png)

* **システム認可済み（License 有効）**：ライセンス署名が正しく有効期限内であることを示します。
* **稼働状態**：右側の ● 正常稼働 は、検証サーバーへの接続（またはオフライン検証）が成功しており改ざんリスクがないことを示します。

<Callout type="info">
**ヒント：** 月に一度は状態を確認してください。赤色になった場合、新規ユーザー作成の制限や一部モデルサービスが停止される可能性があります。
</Callout>
