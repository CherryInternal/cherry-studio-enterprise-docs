# プロバイダー管理

## 📚 エディション概要

Cherry Studio は、ユーザーのタイプに合わせた最適な AI ワークスペース体験を提供します。最大限の制御を求める技術者から、すぐ使える環境を求める企業ユーザー、厳格なコンプライアンスが必要な組織まで、専用のソリューションを用意しています。

現在提供しているコアエディションは次の 3 つです：

1. **Premium (BYOK)**
2. **Express (SaaS)**
3. **Qwen - 通義専版 (Aliyun)**

***

### 1. 🛠️ Premium (BYOK)

> **キーワード：** 柔軟性、自己キー、ローカルモデル、プライバシー重視

Premium は Cherry Studio の基盤であり、**開発者、パワーユーザー、API 資源を持つ企業**向けに設計されています。**BYOK (Bring Your Own Key)** を採用しています。

### 主要特長

* **柔軟な接続**：OpenAI、Azure、DeepSeek、Anthropic、Google Gemini など OpenAI 互換のプロバイダーに対応。
* **ローカルモデル対応**：Ollama、vLLM、LM Studio をフルサポート。データはローカルに留まり機密用途に最適。
* **高度なカスタム**：コンテキスト長、温度、システムプロンプトをモデル単位で制御。

### 適したユーザー

* API Key を持つ開発者。
* ローカル LLM を運用するユーザー。
* 複数モデルを 1 つの UI に集約したいユーザー。

***

### 2. 🚀 Express

> **キーワード：** すぐ使える、グローバルモデル、公式価格、特別なネットワーク不要

Express は **効率重視の個人/チーム**向けのゼロ設定ソリューションです。ChatGPT や Claude の登録、海外カード、特殊なネットワーク設定なしで世界最先端モデルを利用できます。

### 主要特長

* **即時利用**：API Key やプロキシ設定が不要。
* **グローバル主要モデル**：**GPT-5**、**Claude 4.5 Sonnet**、**Google Gemini 3 Pro** などを内蔵。
* **公式価格**：上流（OpenAI/Anthropic）と同価格で **上乗せなし**。使った分だけ支払い、残高は期限なし。
* **統一課金**：複数サブスク不要。1 つの残高で全モデルを利用。

### 適したユーザー

* 海外 API Key を取得できない国内ユーザー。
* ネットワーク設定をせずすぐに使いたいユーザー。
* 高性能モデルを必要な時だけ使いたいユーザー。

***

### 3. ☁️ Qwen 通義専版

> **キーワード：** 阿里云百炼、国内コンプライアンス、企業級安定性、深い統合

Qwen 専版は **阿里云百炼 (Aliyun Bailian)** と深く連携した企業向けエディションです。百炼の接続プロトコルをプリセットし、安定・準拠・高速な国内モデルサービスを提供します。

### 主要特長

* **百炼エコシステム統合**：Qwen-Max/Plus/Turbo と CodeQwen 等を標準統合。
* **企業級保証**：阿里云基盤により高並行性と SLA を提供し、国内コンプライアンスに準拠。
* **簡易導入**：阿里云 API Key を 1 つ入力するだけで全モデルを同期・管理。

### 適したユーザー

* データ越境規制が厳しい企業/政府機関。
* 阿里云百炼の既存顧客。
* 通義千問系モデルを主力に使うチーム。

***

### ⚖️ エディション比較ガイド

以下の比較表をご参考ください：

| 特性 | Premium | Express | Qwen |
| -------- | ------------------- | --------------------------- | ----------------- |
| **コンセプト** | 自由な接続、自前リソース | 簡単・グローバル直結 | 国内準拠、阿里生态 |
| **モデル対応** | 無制限（キーに依存） | GPT-5, Claude Opus 4.6, Gemini など | Qwen 全系列 + 百炼ホストモデル |
| **設定難易度** | ⭐⭐⭐（手動 Key/Host） | ⭐（ゼロ設定） | ⭐⭐（Key のみ） |
| **ネットワーク** | ユーザー側で確保 | **特別なネットワーク不要**（加速内蔵） | 国内直結（高速） |
| **課金** | 直接プロバイダーに支払う | **Cherry 統一ウォレット（公式価格）** | 阿里云百炼の請求 |
| **ローカルモデル** | ✅ 対応（Ollama など） | ❌ 非対応 | ✅ 対応（Ollama など） |
| **推奨シーン** | 開発者/ローカル運用/多元統合 | 個人利用/運用不要/最新モデル体験 | 国内企業の本番運用/コンプライアンス |

### どう選ぶべき？

* **Claude 4.5 でコードを書きたいが海外カードがない** 👉 **Express**。
* **国内サーバー必須で通義千問中心** 👉 **Qwen**。
* **Ollama をローカルで使う、または Gemini 公式キーを持っている** 👉 **Premium**。

<Callout type="info">
**💡 ヒント** エディションの違いはプリセットや認証モジュールで、基盤アーキテクチャは同一です。企業管理者の方は、特定エディションの導入パッケージをご相談ください。
</Callout>
