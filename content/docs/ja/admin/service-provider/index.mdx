## 概要

Cherry Studio エンタープライズ版（Express）は、効率重視の個人とチームに **ゼロ設定** の AI ワークスペース体験を提供します。1 社ずつアカウントを登録したり、個別に API Key を申請したり、自分でパラメータを調整したりする必要はなく、主要な LLM にワンストップで接続でき、すぐに使い始められます。

## 主要特長

* **すぐ使える**：ダウンロードしてすぐ利用可能。API Key の入力やプロバイダーの個別設定は不要。
* **ワンストップ集約**：**DeepSeek**、**智譜 GLM**、**Kimi**、**MiniMax**、**Qwen** など複数の主要 LLM を内蔵。1 つの入口で自由に切り替え。
* **公式価格**：課金はモデル元の価格に準拠。**上乗せなし**。使った分だけ支払い、残高は期限なし。
* **統一課金**：1 つの残高ですべてのモデルを利用可能。複数の異なるサブスクリプションを管理する必要はありません。

## 適したユーザー

* 「ゼロ設定」ですぐに作業を始めたい個人とチーム。
* 複数のモデルアカウントや API Key を 1 社ずつ申請・維持したくないユーザー。
* 複数の主要モデル間で柔軟に切り替え、従量課金で使いたいユーザー。
