## 概要

Cherry Studio エンタープライズ版（Express）は、効率重視の個人とチームに **ゼロ設定** の AI ワークスペース体験を提供します。各モデルのアカウント登録不要で、世界の主要 LLM をすぐに利用できます。

## 主要特長

* **すぐ使える**：API Key やプロキシ設定は不要。
* **グローバル主要モデル**：**DeepSeek-R1**、**GLM-4**、**Qwen** などを内蔵。
* **公式価格**：課金はモデル元の料金体系と同一。**上乗せなし**。使った分だけ支払い、残高は期限なし。
* **統一課金**：1 つの残高ですべてのモデルを利用可能。各モデルのサブスクリプションを個別に管理する必要はありません。

## 適したユーザー

* API Key を個別に取得する手間を省きたいユーザー。
* ネットワーク設定をせずにすぐ作業を始めたいユーザー。
* DeepSeek-R1 などの高性能モデルを必要な時だけ使いたいユーザー。
