# 4. グルーピング戦略

適切なグルーピングは、必要なツールへの到達を速くします。

#### グルーピングの考え方

![グループセレクター](/assets/images/zh/admin/assistant-library-group-selector_zh.webp)

アシスタントの作成/編集時、**所属グループ** は複数選択できます。

* **Featured（厳選）:** 会社全体で最も利用頻度の高いアシスタント。
* **Professional（職業）:** 職種専用アシスタント（例：人事面接官、Java アーキテクト）。
* **Tools（ツール）:** 機能系アシスタント（例：翻訳、JSON フォーマッタ）。

<Callout type="info">
**システムロジック:** ここでのグループはクライアントのアシスタントページのタブに対応します。未割り当てだと「全て」から探す必要があるため、少なくとも 1 つ割り当ててください。
</Callout>
