# アシスタントライブラリ

<Callout type="info">
**ナビゲーション：** サイドバー → リソースライブラリ → アシスタントテンプレート
</Callout>

**アシスタントライブラリ** は Cherry Studio Enterprise の「脳」です。管理者は AI 役割（プロンプト）を集中定義・保存・配布できます。プリセット化されたライブラリにより、社員は検証済みの高品質プロンプトをすぐに使えます。

<img src="/assets/images/ja/admin/assistant-library-overview_ja.webp" alt="" />
### コア価値

* **標準化:** 企業としての AI 応答スタイルを統一（サポート返信、コード規約など）。
* **効率向上:** ゼロからプロンプトを書く必要がない。
* **知識の再利用:** 社内専門家の知見をテンプレート化し全員で共有。

<Callout type="info">
**Tip:** 管理者が設定したアシスタントは、全社員のクライアント「発見」または「アシスタント」ページにリアルタイムで同期されます。
</Callout>
