# 2. アシスタントの作成と編集

アシスタントライブラリの中核機能です。AI アシスタントの挙動をゼロから定義できます。

<img src="/assets/images/ja/admin/assistant-library-create-edit_ja.webp" alt="" />
#### 設定パラメータ

**+ 新規アシスタントテンプレート** または **編集** をクリックすると設定画面が開きます：

<table><thead><tr><th width="151.15234375">項目</th><th width="205.03515625">説明</th><th>ベストプラクティス</th></tr></thead><tbody><tr><td><strong>アシスタント名</strong></td><td>アシスタントの一意名。</td><td>「動詞 + 名詞」形式（例：「Web ページ作成」）。</td></tr><tr><td><strong>絵文字</strong></td><td>アシスタントのアイコン。</td><td>職種に合う絵文字を選択（例：🧑‍💻）。</td></tr><tr><td><strong>所属グループ</strong></td><td>クライアントでの分類に利用。</td><td>少なくとも 1 つのグループ（例：オフィス、コード、厳選）。</td></tr><tr><td><strong>プロンプト</strong></td><td><strong>最重要設定</strong>。人格・口調・タスクを定義。</td><td>構造化プロンプト（役割 + 目標 + 制約）。</td></tr><tr><td><strong>説明</strong></td><td>ユーザーに表示する短い紹介。</td><td>1 文で用途を説明（例：「要件を HTML に変換」）。</td></tr></tbody></table>

#### プロンプト作成のコツ

**プロンプト** は以下の構造を推奨します：

```markdown
# Role
あなたは熟練のフロントエンドエンジニアです。

# Goal
HTML/JS/CSS/TailwindCSS で美しい Web ページを作成します。

# Constraints
1. すべてのコードを 1 つの HTML にまとめる。
2. デザインはモダンで簡潔。
3. 長いコードは分割して説明する。
```

<Callout type="info">
**検証方法:** 作成後、クライアントで「アシスタントライブラリ」から選びテスト会話を行い、制約に従うか確認してください。
</Callout>
