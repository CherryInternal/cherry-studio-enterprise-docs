# 1. アシスタント一覧管理

**リソースライブラリ → アシスタントテンプレート** ページでは、公開済みのアシスタント一覧を確認できます。

### 画面機能

* **検索バー:** アシスタント名/説明のキーワードで検索。
* **一覧項目:**
  * **アシスタント:** 名称とアイコン。
  * **グループ:** 分類タグ（複数可）。
  * **説明:** 機能の概要。
* **操作:** 編集と削除の入口。

<img src="/assets/images/ja/admin/assistant-library-list-management-1_ja.webp" alt="" />
### よく使う操作

<img src="/assets/images/ja/admin/assistant-library-list-management-2_ja.webp" alt="" />
右上の **操作 (Actions)** をクリックするとメニューが開きます：

<img src="/assets/images/ja/admin/assistant-library-list-management-3_ja.webp" alt="" />
* **新規アシスタントテンプレート:** 手動で新規作成。
* **テンプレートから追加:** Cherry Studio 公式テンプレートをインポート。

<Callout type="warn">
**注意:** アシスタント削除は不可逆です。既に保存済みのユーザーには影響しませんが、新規ユーザーは取得できなくなります。
</Callout>
