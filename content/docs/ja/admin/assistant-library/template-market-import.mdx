# 3. テンプレートマーケットのインポート

企業の素早い立ち上げを支援するため、Cherry Studio には 100+ の最適化済みテンプレートが用意されています。

### インポート手順

1. 右上の **操作** > **テンプレートから追加** をクリック。

    <img src="/assets/images/ja/admin/assistant-library-template-market-1_ja.webp" alt="" />
2. **閲覧/検索:** モーダル内で「職業」「ビジネス」「ツール」などを閲覧。
3. **プレビュー:** 説明を確認して適合するか判断。

    <img src="/assets/images/ja/admin/assistant-library-template-market-2_ja.webp" alt="" />
4. **追加:** 右側の **+ 追加** をクリック。

### インポート後の調整

インポートしたアシスタントは一覧に表示されます。**二次編集**が可能です。例：

* 「プロダクトマネージャー」助手のプロンプトを自社の開発フローに合わせる。
* 「コミュニティ運営」助手の口調をブランドに合わせる。

<Callout type="info">
**Tip:** テンプレートはプロンプト作成の良い教材です。高品質なテンプレートをインポートし、プロンプト構造を参考にカスタマイズしてください。
</Callout>
