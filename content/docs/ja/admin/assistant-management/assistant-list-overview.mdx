# 1. アシスタント一覧概要

**アシスタント管理** ページでは、企業内のアシスタント一覧を確認できます。能力分布の全体監視に役立ちます。

![](/assets/images/zh/admin/assistant-management-list-overview_zh.png)

* **検索バー:** アシスタント名で検索。
* **能力指標:**
  * **Knowledge Base (KB):** 付与された私有知識ベース数。
  * **MCP:** 有効化されたツール数。
* **操作:** **編集** または **削除**。
