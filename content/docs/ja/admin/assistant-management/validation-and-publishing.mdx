# 4. 検証と公開

全社員に配布する前に、以下の検証を行ってください：

1. **機能テスト:** 管理者視点でチャットを開き、対象アシスタントを選択。
2. **RAG 検証:** 知識ベースにのみ存在する質問を行い、文書引用できるか確認。
3. **ツール検証:** MCP を有効化している場合（Web 等）、`Cherry Studio の最新バージョンを検索して` と送信し、ツール呼び出しを確認。
4. **権限配布:** 問題なければ **[グループ管理]** で対象グループに割り当て。

<Callout type="info">
**ベストプラクティス:** 先に「ベータテストグループ」を作成し、MCP を含む複雑なアシスタントを配布して安定性を確認してから全体展開してください。
</Callout>
