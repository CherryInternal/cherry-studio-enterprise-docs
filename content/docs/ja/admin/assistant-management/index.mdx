# アシスタント管理

**アシスタント (Assistant)** は Cherry Studio Enterprise の中核単位です。一般的なチャットとは異なり、アシスタントは **役割プロンプト**、**知識境界（Knowledge Base）**、**行動能力（MCP Tools）** を事前に設定した仮想社員です。

このモジュールで、標準化された AI 生産性ツールを大量に作成し、ユーザーグループに配布できます。

<Callout type="info">
**コア価値：**

* **すぐ使える：** プロンプト工学を学ばなくても専門アシスタントを利用可能。
* **知識強化：** 企業の私有文書（PDF/Word/Markdown）を紐付けて精密な QA を実現。
* **能力拡張：** MCP により Web 検索、ファイル操作、DB クエリなどを実行可能。
</Callout>
