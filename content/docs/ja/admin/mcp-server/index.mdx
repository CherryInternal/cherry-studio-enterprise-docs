# MCP サーバー

<Callout type="info">
**ナビゲーション：** サイドバー → リソースライブラリ → MCP サーバー
</Callout>

<img src="/assets/images/ja/admin/mcp-server-overview_ja.webp" alt="" />
MCP サーバーを設定すると、アシスタントに以下の能力を付与できます：

* **Web アクセス**：リアルタイムで検索。
* **ファイル操作**：ローカルのコードやログの読み書き。
* **システム連携**：DB、API、社内システムとの接続。

<Callout type="info">
**コア優位性:** 一度設定すれば全員で共有できます。管理者が設定した MCP サーバーは複数アシスタントに割り当て可能で、ツールコードの再実装が不要です。
</Callout>
