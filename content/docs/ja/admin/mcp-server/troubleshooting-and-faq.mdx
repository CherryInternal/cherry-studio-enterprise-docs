# 4. トラブルシューティングと FAQ

#### Q1: 保存時に "JSON Parse Error" が出る

* **原因**：JSON 形式が不正。
* **対処**：末尾のカンマや中国語引用符を確認。**JSON 形式化** で簡易修正できます。

#### Q2: ツール呼び出しがずっと読み込み中/エラーになる

* **原因**：サーバーにランタイム依存がない。
* **対処**：
  1. Cherry Studio を実行しているホストに **Node.js**（`npx` 用）または **Python** があるか確認。
  2. ネットワーク接続を確認。初回 `npx` 実行時は npm からパッケージを取得するためタイムアウトに注意。

#### Q3: MCP が有効か確認する方法

* **確認方法**：
  1. **アシスタント管理** で MCP サーバーをテスト用アシスタントに紐付け。
  2. チャットでツール呼び出し命令（例：「desktop ディレクトリの test.txt を読み取って」）。
