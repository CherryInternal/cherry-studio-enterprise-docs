## 2. コア機能モジュール

### 👥 ユーザーと権限管理

きめ細かなユーザー/権限管理で、企業のセキュリティとコンプライアンスを支援します。

- **ユーザー管理**：従業員アカウントの一括インポート/管理、アクセス権限と利用クォータの設定、利用状況とアクティビティの監視、複数の認証方式（ローカル、SSO）に対応
- **グループ管理**：部門やプロジェクトグループの作成/管理、グループ権限とリソースの一括割り当て、柔軟な組織構造管理

### 🤖 AI サービスとモデル管理

企業内で利用可能な AI モデルを集中管理し、柔軟な利用とコスト制御を実現します。

- **プロバイダー管理**：40+ の AI プロバイダーに標準対応（OpenAI、Azure OpenAI、Google Gemini、Anthropic Claude、百度 ERNIE、阿里通義など）、OpenAI / Gemini / Claude 等の API 形式によるカスタムプロバイダー接続にも対応
- **デフォルトモデル**：プラットフォームの既定アシスタントモデル、トピック命名モデル、翻訳モデルの設定、モデルごとの利用クォータとレート制限の設定

### 📋 アシスタント管理

企業内の AI アシスタントを管理し、ユーザーに事前設定済みの対話アシスタントを提供します。

- **アシスタント一覧**：企業向け AI アシスタントの作成/管理、可視範囲とグループ権限の設定
- **高度な設定**：プロンプト設定と可視性制御、モデル選択とパラメータ調整、知識ベースと MCP ツールのバインド、アシスタントの公開と検証

### 🎭 エージェント管理

企業内の AI エージェントを管理し、より強力な自律タスク実行機能を提供します。

- **エージェント作成と設定**：ビジュアルな作成ウィザード、キャラクター/行動ルールのカスタム、利用可能なツール/API と知識ベースの関連付け
- **エージェントマーケット**：企業内エージェントの共有とテンプレート管理

### 📦 リソースライブラリ

企業の AI リソースを集中管理します。5 つのモジュールで構成されています：

- **エージェントスキル**：エージェントが使用できるスキルモジュールの管理、手動追加またはコミュニティマーケットからダウンロード、グループ単位で利用範囲を制御
- **アシスタントテンプレート**：再利用可能なアシスタント設定テンプレートの管理、マーケットからのインポートまたは手動作成
- **知識ベース**：マルチソースの取り込み（PDF、DOCX、TXT など）、自動解析とスマート分割、ベクトル化検索とリランキング
- **ミニプログラム**：ミニプログラムアプリの追加/管理、アプリ名、アイコン、URL と権限の設定
- **MCP サーバー**：企業 MCP Server のエンドポイントとツールの設定、アクセス権限の設定、ユーザー権限に応じた設定の自動配布

### 📊 利用統計

包括的なデータ分析と利用状況の監視で、管理者がプラットフォームの利用状況を把握できます。

- **利用概要**：トークン消費量とリクエスト数の統計、ユーザー・モデル・期間ごとの多次元分析、トレンドと分布の可視化チャート
- **ランキング**：ユーザー利用量のランキング、リクエスト数とトークン消費量で並び替え、グループ別フィルタリング

### ⚙️ ライセンス管理とシステム設定

- **ライセンス管理**：License 状態と有効期限の確認、ライセンスユーザー数の監視、License の有効化・更新・更新
- **システム設定**：システム基本設定とパラメータ管理、セキュリティポリシー設定、データバックアップと復旧

### ℹ️ このサービスについて

- **アプリケーション情報**：システムバージョンとビルド情報の確認、クライアントと Docker イメージのダウンロードリンク
- **リリースノート**：各バージョンの更新内容とリリースノートの確認
- **テクニカルサポート**：公式テクニカルサポートの連絡先とサービスチャネル
