Cherry Studio 管理コンソールは、企業向けの集中管理プラットフォームです。AI サービス管理、ユーザー権限管理、リソース設定を包括的に提供します。

<img src="/assets/images/ja/admin-demo.webp" alt="管理コンソールデモ" />
## 体験環境

- **管理コンソール URL**: https://admin.demo.cherry-ai.com/
- **体験アカウント**: `admin`
- **体験パスワード**: `password`

## コア機能モジュール

### 👥 ユーザーと権限管理

きめ細かなユーザー/権限管理で、企業のセキュリティとコンプライアンスを支援します。

### 従業員管理

- 従業員アカウントの一括インポート/管理
- アクセス権限と利用クォータの設定
- 利用状況とアクティビティの監視
- 複数の認証方式（ローカル、SSO）に対応

### グループ管理

- 部門やプロジェクトグループの作成/管理
- グループ権限とリソースの一括割り当て
- 柔軟な組織構造管理

### アクセス制御

- 役割ベースのアクセス制御（RBAC）
- きめ細かな機能権限設定
- API アクセス権限管理

### 🤖 AI サービスとモデル管理

企業内で利用可能な AI モデルを集中管理し、柔軟な利用とコスト制御を実現します。

### プロバイダー管理（モデル接続）

**40+ の AI プロバイダーに標準対応**：

- OpenAI（GPT-5.4、GPT-4 など）
- Azure OpenAI
- Google Gemini
- Anthropic Claude
- 国内主要プロバイダー（百度 ERNIE、阿里通義、iFlytek Spark など）

### カスタムプロバイダー設定

複数の API 形式に対応：

- OpenAI Chat Completions API 形式
- OpenAI Response API 形式
- Google Gemini API 形式
- Anthropic Claude API 形式

### モデルポリシー設定

- **既定アシスタントモデル**：共通 AI 対話機能の既定モデル
- **トピック命名モデル**：会話の要約/タイトル生成用モデル
- **翻訳モデル**：クライアント翻訳機能の既定モデル
- **モデル利用制限**：モデルごとのクォータ/レート制限
- **コスト管理**：利用コストと予算の制御

### 🎭 エージェント管理

### エージェント作成と設定

- ビジュアルな作成ウィザード
- エージェントのキャラクター/行動ルールのカスタム
- 利用可能なツール/ API の設定
- 知識ベースの関連付け

### エージェントマーケット

- 企業内エージェントの共有
- エージェントテンプレート管理
- 利用統計と効果分析

### 📚 企業知識ベース管理

### 知識ベースの作成と管理

- **マルチソースの取り込み**：
  - ドキュメントアップロード（PDF、DOCX、TXT、MD、CSV など）
  - Web コンテンツのクロール
  - API データの取り込み
  - データベース接続
- **スマート処理**：
  - 自動解析と構造化
  - インデックス最適化と分割
  - 多言語対応

### ベクトル化とストレージ

- **ベクトル埋め込み**：高度な埋め込みモデルで高品質ベクトル生成
- **ベクトル DB**：内蔵の高性能ベクトル DB と外部 DB 対応
- **検索最適化**：ハイブリッド検索やリランキングに対応

### 知識ベースのテストと最適化

- 検索効果のテストツール
- 回答品質の評価
- 更新とバージョン管理

### 🔗 MCP Server 管理

### MCP Server 設定

- 企業 MCP Server のエンドポイント設定
- MCP ツール/リソースの管理
- アクセス権限とセキュリティポリシーの設定

### 設定配布

- ユーザー権限に応じて自動配布
- ホットアップデート対応
- 設定のバージョン管理

### 📱 ミニプログラム拡張管理

### ミニプログラム設定

- ミニプログラムアプリの追加/管理
- アプリ名、アイコン、URL の設定
- 必要なパラメータと権限の設定

### アプリ連携

- サードパーティアプリ統合
- OAuth 認証設定
- API ゲートウェイ管理

### ⚙️ システム管理と保守

### License 管理

- License 状態と有効期限の確認
- ライセンスユーザー数の利用状況監視
- License 更新とアップグレード

### システム監視

- リアルタイム性能監視
- API 呼び出し統計/分析
- エラーログとアラート管理

### データ管理

- データバックアップと復旧
- データのインポート/エクスポート
- データクリーンアップとアーカイブ
