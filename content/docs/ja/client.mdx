## ダウンロードとログイン

- **クライアントのダウンロード**: https://enterprise.cherryai.com.cn/download

<img src="/assets/images/ja/client-login-express_ja.webp" alt="クライアントログイン — クラウドサービス" />

クライアントはクラウドサービスのアカウントでログインします。以下の 2 つの方法があります：

- **SMS コードでログイン**：電話番号を入力し、確認コードを受け取ってログインします。
- **パスワードでログイン**：登録済みの電話番号とパスワードでログインします。

アカウントをお持ちでない場合は、左下の「新規登録」をクリックして登録できます。

<PremiumOnly>

### プライベート導入版のログイン

<img src="/assets/images/ja/client-login-premium_ja.webp" alt="Premium 版ログイン — プライベートデプロイ" />

プライベート導入版では、以下の情報を入力してログインします：

- **サーバー URL**：企業が自社でデプロイしたサーバーの URL。
- **ユーザー名**：管理者から割り当てられたユーザー名。
- **パスワード**：ログインパスワード。

SSO が設定されている場合は、左下の「SSO ログイン」をクリックすることもできます。

<Callout type="info">
**体験アカウント：** サーバー URL `https://api.demo.cherry-ai.com`、ユーザー名 `user`、パスワード `password`
</Callout>

ログインページの右下にある「プライベートデプロイ」または「SaaS Cloud」をクリックして、ログイン方式を切り替えられます。

</PremiumOnly>

## ランタイム依存

**エージェント**を利用する場合、クライアントを実行するコンピューターに以下のランタイムを事前にインストールしておく必要があります。チャットやナレッジベースなどの機能のみを使う場合は不要です。

**OS ごとにインストールが必要な項目：**

- **Windows ユーザー**：**Python**、**Node.js**、**Git for Windows**
- **macOS / Linux ユーザー**：**Python**、**Node.js**

### Python

ダウンロードページ：https://www.python.org/downloads/

ページはアクセス元の OS を自動判定するため、上部の大きなダウンロードボタンをクリックするだけで対応するインストーラーが取得できます。下のスクリーンショットは macOS の表示例で、Windows からアクセスするとボタンは "Download the latest version for Windows" と表示されます。

<img src="/assets/images/shared/install/python-download.webp" alt="Python ダウンロードページ" />

ダウンロード完了後、インストーラーをダブルクリックして起動し、ウィザードの指示に従って **Continue** / **次へ** を順にクリックしていけばインストールが完了します：

<img src="/assets/images/shared/install/python-installer.webp" alt="Python インストールウィザード" style={{width:'60%'}} />

### Node.js

1. https://nodejs.org/ にアクセスし、**Get Node.js** ボタンをクリックします：

   <img src="/assets/images/shared/install/nodejs-home.webp" alt="Node.js トップページ" />

2. ダウンロードページ上部のドロップダウンから OS（Windows / macOS / Linux）を選択し、下の **Installer** ボタンをクリックしてインストーラーをダウンロードします：

   <img src="/assets/images/shared/install/nodejs-download.webp" alt="Node.js ダウンロードページ" />

### Git for Windows（Windows ユーザーのみ）

macOS / Linux は bash が標準搭載のため、このセクションは省略可能です。

ダウンロードページ：https://git-scm.com/downloads/win

ページ上部の **Click here to download** リンクをクリックして最新版のインストーラーをダウンロードします。インストール時は既定のオプションのままで問題ありません。インストール完了時に **Git Bash** も同時にインストールされます。

<img src="/assets/images/shared/install/git-windows-download.webp" alt="Git for Windows ダウンロードページ" />

インストール完了後、エージェント作成時に **Git Bash** のパスが自動検出されます。パスが空欄の場合（左図）、Git for Windows がまだインストールされていません：

## 主要機能

### 💬 スマート対話

**マルチモデル対応**

- 企業で設定されたすべての AI モデルにアクセス
- モデルの自動推薦と切り替え
- マルチモーダル対話（テキスト、画像、ファイル）

**会話管理**

- 複数セッションのウィンドウ管理
- 会話履歴の保存と検索
- 会話のエクスポートと共有
- トピックの自動命名と分類

### 🤖 エージェントアシスタント

**プリセットエージェント**

- 企業向けにカスタマイズされた専用エージェント
- 部門共有エージェント
- 個人エージェントのコレクション

**エージェント機能**

- 役割に基づくパーソナライズ応答

### 🔧 MCP ツール連携

**ツール利用**

- 企業で設定された MCP ツールにアクセス

### 📱 ミニプログラムアプリ

### 🎨 パーソナライズ機能

**ショートカット操作**

- ショートカットキーのカスタム

**チーム協業**

## 利用シーン

### 日常業務

- **文書作成**: レポート、メール、提案書の作成を AI が支援
- **情報検索**: 社内文書や知識を素早く検索
- **データ分析**: AI によるデータ解釈と洞察

### 研究開発支援

- **コードアシスタント**: コード生成、最適化、デバッグ
- **技術ドキュメント**: 技術計画の作成と文書生成
- **問題解決**: 技術的な診断と解決策の提示

### カスタマーサービス

- **スマート Q&A**: 知識ベースに基づく顧客対応
- **スクリプト生成**: 営業・サポートのトークスクリプト最適化
- **感情分析**: 顧客フィードバックの分析と対応提案

### クリエイティブ

- **コンテンツ制作**: コピーライティング、マーケティング素材の生成
- **創造的な発想**: ブレインストーミングと企画立案
- **多言語対応**: 翻訳とローカライズ

## システム要件

### Windows

- Windows 10 以降
- 4GB RAM（推奨 8GB）
- 500MB の空きディスク容量

### macOS

- macOS 10.15 以降
- 4GB RAM（推奨 8GB）
- 500MB の空きディスク容量

### Linux

- Ubuntu 20.04 以降
- 4GB RAM（推奨 8GB）
- 500MB の空きディスク容量
