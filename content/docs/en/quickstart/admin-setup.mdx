# Admin Console Setup

After deployment, you need to complete initial setup in the admin console.

## Sign in to the Admin Console

1. Open your browser and visit `http://your-server:3680`
2. Sign in with the default admin account:
   - Username: `admin`
   - Password: `admin123` (change immediately after first login)

## Required Setup Steps

### 1. Configure Providers

Add AI model providers so users can access AI capabilities:

1. Go to **Provider Management**
2. Click **Add Provider**
3. Choose a provider type (OpenAI, Anthropic, Azure, etc.)
4. Enter the API Key and configuration details
5. Test the connection and save

Documentation: [Provider Management](/en/admin/service-provider)

### 2. Create Groups

Groups control which model resources users can access:

1. Go to **Group Management**
2. Click **Create Group**
3. Set a group name (for example, "R&D" or "Marketing")
4. Choose the models available to this group

Documentation: [Group Management](/en/admin/group-management)

### 3. Add Users

Create accounts for team members:

1. Go to **User Management**
2. Click **Add User**
3. Fill in user details (username, email, password)
4. Assign the user to a group

Documentation: [User Management](/en/admin/user-management)
