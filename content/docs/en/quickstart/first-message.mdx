# Send Your First Message

Congratulations! You have completed all configurations. Now send your first message to verify everything works.

## Start a Conversation

1. In the client, click **New Chat**
2. Select an available AI model
3. Enter your first message, for example:

```
Hello! Please introduce yourself.
```

4. Press Enter to send

## Verify Success

If you receive a response from the AI, your setup is correct.

## Common Issues

### Unable to connect to the server

- Check whether the server address is correct
- Confirm the service is running (visit `http://your-server:3670/health`)
- Check firewall settings

### No available models

- Confirm the admin has configured providers
- Confirm your user is assigned to the correct group
- Ask the admin to check model permissions for the group

### Error when sending messages

- Check whether the provider API key is valid
- Confirm model quotas are not exhausted
- Check the call logs in the admin console
