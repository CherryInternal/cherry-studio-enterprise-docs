Cherry Studio offers flexible pricing for users and enterprises of all sizes, from individual developers to large organizations.

## Version Selection Guide

### Express vs Premium: Key Differences

**Same functionality**: Express and Premium are identical in features and both include the full enterprise capability set.

**Only two core differences**:

1. **Provider configuration**

   - Express: built-in providers only, out of the box, no configuration required
   - Premium: fully configurable, add custom providers and private models

2. **Pricing model**
   - Express: ¥9,999, no user limit
   - Premium: [Contact us for pricing](/docs/contact)

### How to choose?

- **Choose Express**: want fast deployment, use mainstream LLM services, no need for custom providers
- **Choose Premium**: need private models, self-hosted services, or special provider requirements

## Detailed Plan Descriptions

### Community

---

**Target users**: Individuals, developers<br />
**Deployment**: Single-machine use<br />
**Technical support**: Community support<br />
**License**: AGPL 3.0 open-source license

**Core features**:

- Completely free, no time limit
- Basic chat features for mainstream AI models
- Local deployment with full data control
- Built-in 30+ providers (OpenAI, Anthropic, Google Gemini, domestic LLMs, etc.)
- **Feature-rich but manual configuration**: users must apply for and configure API keys

**Provider usage**:

- Full provider list available, but users must register accounts themselves
- Manual configuration of API Key, Base URL, and other settings
- Best for technical users who are comfortable managing API keys

**Use cases**:

- Personal AI assistant
- Developer learning and trials
- Internal tools for small teams
- Cost-sensitive individual users

---

**Target users**: Startups, small and mid-sized teams<br />
**Deployment**: Private deployment<br />
**Technical support**: Dedicated technical support (annual service fee required)<br />
**Pricing**: ¥9,999, no user limit

**Core features** (same as Premium):

- ✅ Core features: complete AI chat and multi-model support
- ✅ Provider management: built-in mainstream LLM services (no customization)
- ✅ Model management: manage and configure available AI models
- ✅ Employee management: enterprise user accounts and org structure
- ✅ Shared knowledge base: enterprise vector knowledge base and document management

- ✅ Assistant management: custom AI assistant configuration and management

**Use cases**:
