## Edition Selection Guide

### Express vs Premium

Both editions share the **same core enterprise features**. The key differences are in deployment and billing:

| | **Express** (Cloud-Hosted) | **Premium** (Self-Hosted) |
| --- | --- | --- |
| **Deployment** | No deployment needed — sign up and start | Deploy yourself with Docker / Helm |
| **Data Storage** | Hosted on Cherry Studio cloud | Entirely on your own servers |
| **Provider Configuration** | Built-in major LLMs, out of the box | Fully configurable with custom providers and private models |
| **Billing** | Subscription + Token-based consumption | License (perpetual) |
| **Users** | Tiered by subscription plan | Tiered by license |
| **Support** | Online support | Dedicated technical support (annual service fee) |

### How to Choose?

- **Choose Express**: Zero deployment, instant access, mainstream LLMs, pay-as-you-go
- **Choose Premium**: Full data privacy, private model integration, or compliance requirements

## Premium Pricing

Premium uses a **perpetual license** with tiered pricing by headcount:

| Team Size | Description |
| --- | --- |
| 1–50 users | Starter plan |
| 51–200 users | Mid-sized team |
| 200+ users | Large organization, custom quote |

</Callout>

### Annual Support Fee (Optional)

After purchasing a Premium license, you can optionally add an annual support fee, which includes:

- 7x24 technical support
- Remote deployment assistance
- Custom requirement consultation
