## Version Selection Guide

### Express vs Premium

**Same functionality**: Express and Premium are identical in features and both include the full enterprise capability set.

**Only two core differences**:

1. **Provider configuration**

   - Express: built-in providers only, out of the box, no configuration required
   - Premium: fully configurable, add custom providers and private models

2. **Pricing model**
   - Express: ¥9,999, no user limit
   - Premium: [Contact us for pricing](/docs/contact)

### How to choose?

- **Choose Express**: want fast deployment, use mainstream LLM services, no need for custom providers
- **Choose Premium**: need private models, self-hosted services, or special provider requirements
