## Step 3: Add a Provider (Premium Only)

<Callout type="warn">
This step applies to **Premium** only. Express comes with built-in providers and requires no manual setup.
</Callout>

Log in to the admin console and go to "Provider Management" to add AI providers (such as OpenAI or Anthropic) and configure API keys.

<Callout type="info">
  If you need the first-login admin account details, check the [installation and deployment guide](/docs/setup) first. For cloud app deployment, use the account information provided on the app settings page.
</Callout>

<img src="/assets/images/en/admin/provider-management-standard-1_en.webp" alt="Add provider" />

<Callout type="info">
  [View the full provider configuration guide →](/docs/admin/service-provider)
</Callout>
