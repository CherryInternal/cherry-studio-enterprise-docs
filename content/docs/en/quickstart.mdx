## Step 3: Configure the System

<Steps>
<Step>

### Add a Provider

Log in to the admin console and go to "Provider Management" to add AI providers (such as OpenAI or Anthropic) and configure API keys.

<Callout type="info">
  If you need the first-login admin account details, check the <DynamicLink href="/[lang]/docs/setup">installation and deployment guide</DynamicLink> first. For cloud app deployment, use the account information provided on the app settings page.
</Callout>

<img src="/assets/images/zh/admin/provider-management-standard-1_zh.webp" alt="Add provider" />
<Callout type="info">
  <DynamicLink href="/[lang]/docs/admin/service-provider">View the full provider configuration guide →</DynamicLink>
</Callout>

</Step>

<Step>

### Set the Default Model

Go to "Model Management" to configure available models for the provider and set a default model.

<img src="/assets/images/zh/admin/default-model-settings-overview_zh.webp" alt="Set default model" />
<Callout type="info">
  <DynamicLink href="/[lang]/docs/admin/model-settings">View the full model configuration guide →</DynamicLink>
</Callout>

</Step>

<Step>

### Add Users

Go to "User Management" and create accounts for team members.

<img src="/assets/images/zh/admin/user-management-add-user-modal_zh.webp" alt="Add users" />
<Callout type="info">
  <DynamicLink href="/[lang]/docs/admin/user-management">View the full user management guide →</DynamicLink>
</Callout>

</Step>

<Step>

### Download the Client

Download and install the Cherry Studio Enterprise client.

  <a href="https://github.com/CherryHQ/cherry-studio/releases" target="_blank" style={{display:'inline-flex',alignItems:'center',gap:'6px',height:'36px',padding:'0 14px',backgroundColor:'#1a6eff',color:'white',borderRadius:'6px',fontSize:'14px',fontWeight:500,textDecoration:'none'}}>

    <span style={{lineHeight:1}}>Download Client</span>
  </a>
</div>

</Step>

<Step>

### Sign In and Send Your First Message

Sign in to the client with the account created by the admin to start using all features.

<img src="/assets/images/zh/client-chat-demo.webp" alt="Client chat" />
<Callout type="info">
  <DynamicLink href="/[lang]/docs/client">View the full client guide →</DynamicLink>
</Callout>

</Step>

<Step>

### Done

Congratulations! You have completed deployment and configuration for Cherry Studio Enterprise.

</Step>
</Steps>
