The **Knowledge Base** is the core component of Cherry Studio Enterprise's **RAG (Retrieval-Augmented Generation)**. It acts as an "external brain" for LLMs, converting private documents, webpages, and notes into vector storage.

When users interact with assistants, the system first retrieves relevant information from the knowledge base and feeds it as context to the model, enabling fact-based answers, reducing hallucinations, and improving data security.
