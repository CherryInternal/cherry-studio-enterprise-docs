# 3. Edit and Maintain

#### Edit User Information <a href="#heading8ca0746f50e147119c6c47d6dded5dd2-bian-ji-yong-hu-xin-xi-0" id="heading8ca0746f50e147119c6c47d6dded5dd2-bian-ji-yong-hu-xin-xi-0"></a>

![Edit user modal](/assets/images/zh/admin/user-management-edit-user-modal_zh.png)

To change group permissions or reset a password:

1. Find the user in the list.
2. Click the **⋮** (more) icon in the **Actions** column.
3. Select **Edit**.
4. **Common actions**:
   * **Reset password**: Enter a new value in the "New Password" field.
   * **Permission change**: Update **Group** and changes take effect immediately.
   * **Account suspension**: Turn off **Active status** to revoke login access immediately.
5. Click **Update User** to save.

#### Delete Users <a href="#heading8ca0746f50e147119c6c47d6dded5dd2-shan-chu-yong-hu-0" id="heading8ca0746f50e147119c6c47d6dded5dd2-shan-chu-yong-hu-0"></a>

For offboarded employees or obsolete test accounts:

1. Click the **⋮** icon in the **Actions** column.
2. Select **Delete** (red option).

![](/assets/images/zh/admin/user-management-edit-user-1_zh.png)

3. Confirm the deletion in the confirmation dialog.

![](/assets/images/zh/admin/user-management-edit-user-2_zh.png)

<Callout type="error">
**Warning:** Deletion is **irreversible**. After deletion, the user's personal knowledge base settings and chat history may not be recoverable.
</Callout>
