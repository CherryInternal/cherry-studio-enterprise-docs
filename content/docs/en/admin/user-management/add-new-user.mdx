# 2. Add New User

#### Workflow <a href="#heading8ca0746f50e147119c6c47d6dded5dd2-cao-zuo-liu-cheng-0" id="heading8ca0746f50e147119c6c47d6dded5dd2-cao-zuo-liu-cheng-0"></a>

1. In the top-right of the User Management page, click **+ Add User**.
2. In the drawer/modal, complete the configuration:

![Add user modal](/assets/images/zh/admin/user-management-add-user-modal_zh.png)

#### Parameter Details <a href="#heading8ca0746f50e147119c6c47d6dded5dd2-pei-zhi-can-shu-xiang-jie-0" id="heading8ca0746f50e147119c6c47d6dded5dd2-pei-zhi-can-shu-xiang-jie-0"></a>

<table><thead><tr><th width="105.7265625">Field</th><th width="76.95703125">Required</th><th width="139.546875">Format</th><th>Description</th></tr></thead><tbody><tr><td><strong>Group</strong></td><td><strong>Yes</strong></td><td>Dropdown</td><td><strong>Core setting</strong>. Determines which models (Model Providers) and assistants the user can access.</td></tr><tr><td><strong>Username</strong></td><td><strong>Yes</strong></td><td>At least 3 characters</td><td>Recommend using name or employee ID; must be unique.</td></tr><tr><td><strong>Email</strong></td><td><strong>Yes</strong></td><td>Valid email format</td><td>Must be valid for notifications.</td></tr><tr><td><strong>Password</strong></td><td><strong>Yes</strong></td><td>At least 6 characters</td><td>Initial login password.</td></tr><tr><td><strong>Phone</strong></td><td>No</td><td>Numeric format</td><td>Optional contact info.</td></tr><tr><td><strong>Active status</strong></td><td><strong>Yes</strong></td><td>Toggle (on by default)</td><td>If off, the user cannot sign in until activated by an admin.</td></tr></tbody></table>

3. Click **Create User** to finish.

<Callout type="info">
**Tip:** For security, notify users to change the initial password after first login.
</Callout>
