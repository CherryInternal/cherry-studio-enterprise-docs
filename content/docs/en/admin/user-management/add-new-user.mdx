# 2. Add New User

### Workflow <a href="#heading8ca0746f50e147119c6c47d6dded5dd2-cao-zuo-liu-cheng-0" id="heading8ca0746f50e147119c6c47d6dded5dd2-cao-zuo-liu-cheng-0"></a>

1. In the top-right of the User Management page, click **+ Add User**.
2. In the drawer/modal, complete the configuration:

<img src="/assets/images/en/admin/user-management-add-user-modal_en.webp" alt="Add user modal" />
### Parameter Details <a href="#heading8ca0746f50e147119c6c47d6dded5dd2-pei-zhi-can-shu-xiang-jie-0" id="heading8ca0746f50e147119c6c47d6dded5dd2-pei-zhi-can-shu-xiang-jie-0"></a>

<table><thead><tr><th width="105.7265625">Field</th><th width="76.95703125">Required</th><th width="139.546875">Format</th><th>Description</th></tr></thead><tbody><tr><td><strong>Group</strong></td><td><strong>Yes</strong></td><td>Dropdown</td><td><strong>Core setting</strong>. Determines which models (Model Providers) and assistants the user can access.</td></tr><tr><td><strong>Username</strong></td><td><strong>Yes</strong></td><td>At least 3 characters</td><td>Recommend using name or employee ID; must be unique.</td></tr><tr><td><strong>Password</strong></td><td><strong>Yes</strong></td><td>At least 6 characters</td><td>Initial login password.</td></tr><tr><td><strong>Group Admin</strong></td><td>No</td><td>Toggle (off by default)</td><td>When enabled, the user is set as an admin of their group.</td></tr></tbody></table>

3. Click **Create User** to finish.

<Callout type="info">
**Tip:** For security, notify users to change the initial password after first login.
</Callout>
