When your organization needs to create a large number of user accounts at once, adding them one by one is inefficient. The batch import feature supports importing multiple users via **Excel (.xlsx)** or **CSV (.csv)** files, with automatic creation of new groups.

<Callout type="info">
**Entry point:** The **Import** button in the top-right corner of the User Management page, or the Group Management page.
</Callout>

## Workflow

The import process has 3 steps:

### Step 1: Upload File

<img src="/assets/images/en/admin/user-management-import-step1_en.webp" alt="Batch Import - Upload File" />

1. Click **Download Excel Template** or **Download CSV Template** to get the standard template.
2. Fill in user data following the template format.
3. Drag and drop the completed file to the upload area, or click to select.
4. Click **Next** to proceed to data preview.

### Step 2: Data Preview

After parsing, the system displays a preview table for verification. Format errors or missing required fields are highlighted.

### Step 3: Execute Import

Click execute to batch-create users and groups. A results summary (successful, skipped, failed counts) is shown upon completion.

<Callout type="warn">
**Important notes:**
* Do not modify the header row in the template.
* Existing usernames can be skipped or updated.
* Non-existent groups are auto-created (admin permission required).
* Test with a small dataset first before bulk importing.
</Callout>
