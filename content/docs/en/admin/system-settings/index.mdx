# System Settings

Correct configuration is critical for the following experiences:

* **Model connectivity**: Resolve connectivity to OpenAI/Anthropic in restricted networks.
* **Knowledge base accuracy**: Improve PDF parsing with OCR services.
* **Real-time web access**: Enable assistants to search real-time news.
* **Performance optimization**: Manage sync frequency and local storage.
