# 1. Basic Settings

#### Configuration

1. Go to **System Settings** > **Basic Settings**.
2. Adjust sync parameters as needed.

![](/assets/images/zh/admin/system-settings-basic_zh.webp)

#### Parameter Details

<table><thead><tr><th width="115.4765625">Parameter</th><th width="219.5859375">Description</th><th>Recommended</th></tr></thead><tbody><tr><td><strong>Client data sync interval</strong></td><td>Controls how often local data (chat history, settings) syncs to server. Unit: minutes.</td><td><strong>Default: 1</strong><br />• <strong>Low frequency (5-10)</strong>: For low-spec devices, reduces background load.<br />• <strong>High frequency (1)</strong>: Minimizes data loss risk.</td></tr></tbody></table>

<Callout type="info">
**Tip:** Changes are saved automatically. In poor networks, consider increasing the interval.
</Callout>
