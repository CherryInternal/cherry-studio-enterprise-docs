## Status Metrics

<table><thead><tr><th width="141.5234375">Metric</th><th>Meaning</th></tr></thead><tbody><tr><td><strong>Total files / size</strong></td><td>All files under the data directory.</td></tr><tr><td><strong>Valid files</strong></td><td>Files referenced by the knowledge base or active chats. <strong>The system protects these files from deletion.</strong></td></tr><tr><td><strong>Invalid files</strong></td><td>Orphan files whose DB records were deleted but files remain on disk.</td></tr></tbody></table>

## Cleanup

1. Click **Refresh stats** to scan disk usage.
2. Review **Invalid files** count and size.
3. Click **Clean invalid files** to run cleanup.

<img src="/assets/images/en/admin/system-settings-cache_en.webp" alt="" />
## ⚠️ Risk Warning

<Callout type="error">
**Data safety warning**:

* Cleanup will **permanently delete** files marked as invalid.
* This operation is irreversible.
* Although the system identifies invalid files via DB comparison, **we strongly recommend** backing up the Cherry Studio data directory before cleanup to avoid loss due to edge cases.
</Callout>
