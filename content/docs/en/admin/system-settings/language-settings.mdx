---
title: "Language Settings"
description: "Set the display language for the admin panel. Supports auto-detection, Simplified Chinese, English, and Japanese."
---

## How to Set

1. Go to **System Settings** > **Language**.
2. Select the desired language from the **Select Language** dropdown.

<img src="/assets/images/en/admin/system-settings-language_en.webp" alt="Language Settings" />

## Available Languages

| Option | Description |
|---|---|
| **Follow System** | Automatically matches the browser's language. |
| **简体中文** | Display the admin panel in Simplified Chinese. |
| **English** | Display the admin panel in English. |
| **日本語** | Display the admin panel in Japanese. |

The language setting only affects the current browser. Changes take effect immediately without refreshing the page.
