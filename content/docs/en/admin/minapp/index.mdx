# Mini Program Management

The **Mini Program Management** module is a core feature for building a "one-stop AI workspace" in Cherry Studio Enterprise. Admins can integrate external AI tools, internal systems, or common web services into the client's sidebar so employees can access resources without switching browsers.

This feature supports mainstream AI web apps like DeepSeek and Kimi, and also allows embedding internal document libraries or LLM WebUIs.
