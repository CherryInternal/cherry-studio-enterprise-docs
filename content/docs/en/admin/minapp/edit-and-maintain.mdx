# 3. Edit and Maintain

For **custom mini programs**, admins can adjust configurations at any time.

<img src="/assets/images/en/admin/mini-program-edit-1_en.webp" alt="" />
### Update Info

1. Find the target mini program (type: Custom).
2. Click **Edit** in the actions column.
3. Update **URL** (e.g., service migration) or replace the **Logo**.
4. Click **Update Mini Program** to apply.

### Delete Mini Program

* Click the red **Delete** button to permanently remove it.
* **Note:** Deletion takes effect immediately; the entry disappears from all employee clients.
