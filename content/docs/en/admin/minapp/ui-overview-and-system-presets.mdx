# 1. UI Overview and System Presets

Navigate to **Resource Library → Mini Programs** to see all available apps in the enterprise environment.

#### UI Layout

<img src="/assets/images/en/admin/mini-program-overview_en.webp" alt="" />
The list includes two categories:

1. **System presets:** Officially curated mainstream AI services (DeepSeek, ChatGPT, Qwen, etc.). The URLs are maintained by Cherry Studio; admins only enable/disable them.
2. **Custom:** Internal or third-party links added by admins.

<Callout type="info">
**Why can’t some apps be deleted?** System preset mini programs are hard-coded for stability. They can be **disabled** (hidden) but not deleted.
</Callout>

#### Quick Enable/Disable

In the **Enabled** column on the right, toggle whether the app appears in the client sidebar.

* **ON:** All employees will see the icon after restarting the client.
* **OFF:** The app is hidden but configuration is kept.
