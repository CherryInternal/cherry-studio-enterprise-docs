title: "Add Custom Mini Program"
description: "Package any Web URL as a mini program with name, icon, and group settings, then add it to the client."
---

import { Callout } from 'fumadocs-ui/components/callout'

## Steps

1. Click **+ Add Mini Program** in the top right.

<img src="/assets/images/en/admin/mini-program-add-custom_en.webp" alt="" />
2. In the dialog, complete the configuration:

<table><thead><tr><th width="91.7421875">Module</th><th width="94.125">Field</th><th>Description</th></tr></thead><tbody><tr><td><strong>Visual identity</strong></td><td><strong>Upload logo</strong></td><td>Use <strong>PNG</strong> or <strong>WebP</strong> with transparent background if possible. Recommended size <code>200x200px</code>, max <code>200KB</code>.</td></tr><tr><td></td><td><strong>Name</strong></td><td>Text shown under the icon (keep short for aesthetics).</td></tr><tr><td><strong>Core config</strong></td><td><strong>URL</strong></td><td><strong>Must include protocol</strong> (e.g., <code>https://</code>).<br />Example: <code>https://docs.cherry-ai.com</code></td></tr><tr><td><strong>Style tweaks</strong></td><td><strong>Border</strong></td><td>Adds a subtle border around the icon; useful if logo color matches sidebar background.</td></tr><tr><td></td><td><strong>Background color</strong></td><td>Set a background color for the logo (supports hex like <code>#FFFFFF</code>). For transparent PNGs, this improves visibility.</td></tr></tbody></table>

3. Click **Add Mini Program** to save.

<Callout type="warn">
**URL restriction:** Use secure links starting with `https://`. Due to client security policies, regular `http://` links may be blocked or fail to load.
</Callout>
