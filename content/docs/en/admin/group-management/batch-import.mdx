## Overview

Batch import allows admins to create multiple groups at once by uploading an Excel or CSV file — ideal for quickly setting up an organization structure.

## Steps

1. Go to Admin Console → Group Management
2. Click the **Import** button in the top-right corner
3. Download the standard template (Excel or CSV format) from the dialog
4. Fill in the group data according to the template rules
5. Upload the completed file and click **Next Step** to start the import

<img src="/assets/images/en/admin/group-management-batch-import_en.webp" alt="Batch Import Groups" />

### Template Fields

| Field Name | Required | Description |
| --- | --- | --- |
| **name** | Yes | Group name, unique identifier |
| **description** | No | Group description |

## Notes

- The first row is the header — do not modify it
- Existing group names can be skipped or updated
