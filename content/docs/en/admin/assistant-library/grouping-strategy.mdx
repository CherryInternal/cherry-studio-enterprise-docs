## Grouping Logic

<img src="/assets/images/en/admin/assistant-library-group-selector_en.webp" alt="Group selector" />
When creating or editing assistants, the **Group** field supports multiple selection.

* **Featured:** Put the most frequently used assistants here.
* **Professional:** Role-specific assistants (e.g., HR interviewer, Java architect).
* **Tools:** Utility assistants (e.g., translator, JSON formatter).
* **Other groups:** Select as needed.

<Callout type="info">
**System logic:** These groups map directly to tabs on the client assistant page. If an assistant has no group, users may need to search in "All". Assign at least one group.
</Callout>
