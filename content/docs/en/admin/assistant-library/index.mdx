The **Assistant Library** is the "brain" of Cherry Studio Enterprise. Admins can centrally define, store, and distribute standardized AI roles (prompts). With a preset library, employees can use proven high-quality prompts without each person becoming a prompt engineer.

<img src="/assets/images/en/admin/assistant-library-overview_en.webp" alt="" />
### Core Value

* **Standardized delivery:** Unify AI response style (e.g., support replies, code conventions).
* **Efficiency boost:** Employees can use assistants instantly without writing prompts from scratch.
* **Knowledge reuse:** Convert expert experience into prompt templates shared company-wide.

<Callout type="info">
**Tip:** Assistants configured in the admin console are synced in real time to all employees' Cherry Studio client "Discover" or "Assistants" pages.
</Callout>
