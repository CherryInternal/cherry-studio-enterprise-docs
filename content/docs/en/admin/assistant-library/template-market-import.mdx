# 3. Template Market Import

To help enterprises get started quickly, Cherry Studio includes 100+ optimized assistant templates.

#### Import Flow

1. Click **Actions** > **Add from templates** in the top right.

    ![](/assets/images/zh/admin/assistant-library-template-market-1_zh.png)
2. **Browse/Search:** In the modal, browse templates across categories like "Professional", "Business", "Tools".
3. **Preview:** Read the description to confirm fit.

    ![](/assets/images/zh/admin/assistant-library-template-market-2_zh.png)
4. **Add:** Click **+ Add** on the right.

#### Post-import Adjustments

Imported assistants appear in your list. You can **edit** them, for example:

* Modify the "Product Manager" prompt to fit your internal process.
* Adjust the "Community Ops" assistant tone to match your brand.

<Callout type="info">
**Tip:** The template library is a great way to learn prompt writing. Import a high-quality template, inspect its prompt, and customize it.
</Callout>
