## Unauthorized

<img src="/assets/images/en/admin/authorization-import_en.webp" alt="Import license" />
* **Trial Period**: The system is in trial mode with full functionality available. Import a valid License file before the trial expires to continue using all features.

## License Expired

<img src="/assets/images/en/admin/authorization-expired_en.webp" alt="License expired" />
* **Trial Expired**: The trial period has ended and system features are limited. Please import a valid License file to restore full functionality.

## Authorization Failed

<img src="/assets/images/en/admin/authorization-validation-failed_en.webp" alt="Validation failed" />
* **License validation failed**: The license file signature is invalid, expired, or does not match the current deployment environment.

## Authorization Successful

<img src="/assets/images/en/admin/authorization-valid_en.webp" alt="Authorized and running" />
* **System licensed (License valid)**: the license file signature is correct and within validity.
* **Runtime status**: the ● Running indicator on the right shows the system has successfully connected to the validation server (or passed offline validation) with no tampering risk.

<Callout type="info">
**Tip:** Check this status monthly. If it turns red, the system may limit new users or suspend some advanced model services.
</Callout>
