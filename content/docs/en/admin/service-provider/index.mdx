## Overview

Cherry Studio Enterprise (Express) gives individuals and teams a zero-configuration AI workspace. No need to register accounts provider by provider, apply for separate API keys, or tune parameters yourself — get one-stop access to mainstream models that work right out of the box.

## Core Features

* **Out of the box**: No API keys to fill in, no providers to configure one by one.
* **One-stop aggregation**: Built-in access to multiple mainstream models such as **DeepSeek**, **Zhipu GLM**, **Kimi**, **MiniMax**, and **Qwen** — switch freely from a single entry point.
* **Official pricing**: Billing stays consistent with the upstream model vendors. **No markup**. Pay as you go; balance never expires.
* **Unified billing**: One account balance covers all models — no need to juggle multiple separate subscriptions.

## Ideal For

* Individuals and teams who want to start working immediately with zero configuration.
* Users who don't want to apply for and maintain multiple model accounts and API keys.
* Users who need to switch flexibly between mainstream models and pay by usage.
