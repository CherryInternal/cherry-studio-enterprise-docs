## Overview

Cherry Studio Enterprise (Express) gives individuals and teams a zero-configuration AI workspace. No need to register for individual model accounts, apply for separate API keys, or configure special networks — use top models directly.

## Core Features

* **Out of the box**: No API keys or proxy setup required.
* **Top models**: Built-in **DeepSeek-R1**, **GLM-4**, **Qwen-Max**, **Kimi**, and more.
* **Official pricing**: Billing matches original model providers exactly. **No markup**. Pay as you go; balance never expires.
* **Unified billing**: One account balance covers all models — no need to juggle multiple subscriptions across various LLM providers.

## Ideal For

* Users who cannot apply for overseas API keys.
* Writers who want to start immediately without dealing with network setup.
* Users who need premium models (e.g. DeepSeek-R1) occasionally without monthly subscriptions.
