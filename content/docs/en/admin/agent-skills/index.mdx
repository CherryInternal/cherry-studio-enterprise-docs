# Agent Skills

**Agent Skills** are reusable capability modules in the Resource Library, designed for agents. Admins can add skills via **manual upload** or **marketplace download**. Once added, skills can be bound to agents during creation or editing.

<Callout type="info">
**Navigation:** Sidebar → Resource Library → Agent Skills (first tab)
</Callout>

<img src="/assets/images/en/admin/agent-skills-list_en.webp" alt="Agent Skills List" />

## Skills List

The skills list displays all added agent skills with the following information:

<table><thead><tr><th width="100">Field</th><th>Description</th></tr></thead><tbody><tr><td><strong>Name</strong></td><td>The skill's identifier name.</td></tr><tr><td><strong>Group</strong></td><td>The group the skill belongs to, determining which user groups can access it.</td></tr><tr><td><strong>Description</strong></td><td>A description of the skill's functionality.</td></tr><tr><td><strong>Actions</strong></td><td>Edit, view details, delete, and other operations.</td></tr></tbody></table>

Use the search box to quickly find skills by name.

## Adding Skills

Click the **Add** button in the top-right corner. Two methods are available:

<img src="/assets/images/en/admin/agent-skills-add-menu_en.webp" alt="Add Skills Menu" />

### Manual Upload

For custom enterprise-built or third-party skills. Upload skill files (ZIP or Markdown format) and fill in metadata such as name and description.

### Download from Marketplace

Opens the skill marketplace browser where you can browse and download pre-built skills from the official Cherry Studio marketplace. Marketplace skills are officially verified and ready to use.

<Callout type="info">
**Workflow:** Add skills in Resource Library → Enable skills in the agent's "Skills" tab → The agent can now use the skill's capabilities at runtime.
</Callout>
