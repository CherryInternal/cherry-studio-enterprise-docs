Click on any order to view its full details.

<img src="/assets/images/en/admin/order-management-detail_en.webp" alt="Order Details" />

## Order Information

| Field | Description |
| --- | --- |
| **Order No.** | Unique order number. |
| **Company** | Name of the ordering company. |
| **Created At** | When the order was created. |
| **Paid At** | When payment was completed. |
| **Payment Method** | Bank Transfer or Alipay. |

## Product Information

| Field | Description |
| --- | --- |
| **Product Name** | The purchased product, e.g. "Token Recharge". |
| **Product Code** | System code for the product, e.g. `token-recharge`. |
