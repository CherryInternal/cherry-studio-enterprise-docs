By configuring MCP servers, you can grant assistants the following capabilities:

* **Web access**: Search the web in real time.
* **File operations**: Read local codebases or write log files.
* **System integration**: Connect databases, APIs, or internal systems.

<Callout type="info">
**Core advantage:** Configure once, share across all. Admin-configured MCP servers can be assigned to multiple assistants without rewriting tool code.
</Callout>
