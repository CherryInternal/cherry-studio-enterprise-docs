**Path:** Admin console sidebar > **MCP Servers**

This page is the MCP console where admins can view all registered tool services.

<img src="/assets/images/en/admin/mcp-server-list-management_en.webp" alt="" />
### Core Fields

<table><thead><tr><th width="94.671875">Field</th><th>Description</th></tr></thead><tbody><tr><td><strong>Name</strong></td><td>Display name (e.g., <code>mcp-server-chart</code>).</td></tr><tr><td><strong>Group</strong></td><td>The group the server belongs to, determining which user groups can access it.</td></tr><tr><td><strong>ID</strong></td><td>Unique internal identifier (e.g., <code>antv-chart</code>).</td></tr><tr><td><strong>Type</strong></td><td>Protocol type, e.g., <code>stdio</code> (standard input/output).</td></tr><tr><td><strong>Description</strong></td><td>A description of the server's functionality.</td></tr><tr><td><strong>Associated Assistants</strong></td><td><strong>[Key metric]</strong> Number of assistants using this server.</td></tr><tr><td><strong>Actions</strong></td><td>Supports <strong>View</strong> details, <strong>Edit</strong> configuration, or <strong>Delete</strong> the server.</td></tr></tbody></table>

<Callout type="warn">
**Deletion risk:** Before deleting, ensure **Associated assistants** is `0`. Deleting a server in use will cause tool calls to fail.
</Callout>
