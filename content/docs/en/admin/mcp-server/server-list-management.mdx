# 1. Server List Management

**Path:** Admin console sidebar > **MCP Servers**

This page is the MCP console where admins can view all registered tool services.

<img src="/assets/images/zh/admin/mcp-server-list-management_zh.webp" alt="" />
#### Core Fields

<table><thead><tr><th width="94.671875">Field</th><th>Description</th></tr></thead><tbody><tr><td><strong>Name</strong></td><td>Display name (e.g., <code>Playwright</code>).</td></tr><tr><td><strong>ID</strong></td><td>Unique internal identifier (e.g., <code>JfzwZ...</code>).</td></tr><tr><td><strong>Type</strong></td><td>Protocol type, default <code>stdio</code> (standard input/output).</td></tr><tr><td><strong>Associated assistants</strong></td><td><strong>[Key metric]</strong> Number of assistants using this server.</td></tr><tr><td><strong>Actions</strong></td><td>Supports <strong>Edit</strong> or <strong>Delete</strong>.</td></tr></tbody></table>

<Callout type="warn">
**Deletion risk:** Before deleting, ensure **Associated assistants** is `0`. Deleting a server in use will cause tool calls to fail.
</Callout>
