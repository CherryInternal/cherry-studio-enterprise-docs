# Default Model Settings

**Path:** Admin console sidebar > **Default Models**

On this page, admins define the **global fallback models** for Cherry Studio Enterprise across different scenarios. Proper configuration balances user experience (speed, quality) with token costs.

![](/assets/images/zh/admin/default-model-settings-overview_zh.webp)

<Callout type="info">
**Prerequisite:** The dropdown lists only models that are added and enabled in **[Model Management]**. If you don't see a model, add it there first.
</Callout>
