# 1. Default Assistant Model

**Definition**: When a user starts a new chat or creates an assistant without specifying a model, the system uses this default.

* **Business impact**: This is the first impression of user experience. It determines the intelligence for casual queries.
* **Recommendations**:
  * **Experience-first**: Choose a strong flagship model like `DeepSeek-R1` or `Gemini-3-Pro` for high-quality first impressions.
  * **Cost-effective**: For high concurrency, choose a "sweet spot" model like `DeepSeek-V3.2` or `Gemini-3-flash` to balance speed and cost.
