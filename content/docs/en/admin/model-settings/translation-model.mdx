# 3. Translation Model

**Definition**: The underlying model used by Cherry Studio's built-in translation features (selection translation, message translation).

* **Business impact**: Affects multilingual collaboration accuracy.
* **Recommendation**:
  * Choose models strong in multilingual training.
  * **Recommended**: `DeepSeek-V3.2` (excellent Chinese-English translation) or `GPT-5`.
