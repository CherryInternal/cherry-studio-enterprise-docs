To help you decide quickly, here are recommended setups for different organization types:

## 🚀 Plan A: Performance First (R&D/Algorithm teams)

* **Default assistant**: `DeepSeek-Reasoner` (deep reasoning for complex code problems)
* **Topic naming**: `Qwen-Turbo` (fast title generation)
* **Translation model**: `DeepSeek-V3.2`

## ⚖️ Plan B: Balanced Cost (General office/Admin)

* **Default assistant**: `DeepSeek-V3.2` (balance speed and intelligence, much cheaper than R1)
* **Topic naming**: `Qwen-2.5-7B` (low-cost local or cloud model)
* **Translation model**: `DeepSeek-V3.2`

## 💰 Plan C: Ultra-fast Response (Customer service/Sales)

* **Default assistant**: `GPT-5.1-mini` or `DeepSeek-Lite` (instant response)
* **Topic naming**: `GPT-5.1-mini`
* **Translation model**: `GPT-5.1-mini`

***

## FAQ

**Q: What happens if I delete a model used here?**

A: If you delete a default model in **Model Management**, the system may fall back to built-in defaults or error. Switch to another model before deleting.

**Q: What if the topic naming model is too weak?**

A: If you choose a model with poor instruction following (e.g., early 1B models), it might use the prompt itself as the title or generate gibberish, making the conversation list hard to read.
