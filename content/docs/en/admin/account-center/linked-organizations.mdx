In Account Center, click the **Organizations** tab to view all organizations you have joined or created, switch between them, or join a new one.

<img src="/assets/images/en/admin/account-center-organizations_en.webp" alt="Organizations" />

## Organization List

All organizations linked to the current account are listed as cards. Each card shows:

- **Organization name** and **status** (e.g., "Active")
- **Role** (Owner / Administrator / Member)
- **Enter Admin** button — jump directly to that organization's admin console

<Callout type="info">
Only owners and administrators can enter the admin console. Regular members may see the "Enter Admin" button as unavailable.
</Callout>

## Join with an Enterprise Invite Code

If you've received an invite link or enterprise invite code from an organization administrator:

1. Paste the enterprise invite code or invite link into the input box.
2. Click **Join**.
3. Once joined, the new organization appears in the list above.

<Callout type="info">
For details on how administrators invite members, see <DynamicLink href="/[lang]/docs/admin/user-management/invite-members">Member Invitation</DynamicLink>.
</Callout>

## Create a New Organization

Click **+ Create Organization** at the bottom of the list to create a new organization. You'll become the owner and need to complete enterprise verification and the payment activation flow.

<Callout type="info">
For the full creation flow (Register → Create Organization → Select Plan → Enterprise Verification → Payment), see <DynamicLink href="/[lang]/docs/quickstart">Quickstart</DynamicLink>.
</Callout>
