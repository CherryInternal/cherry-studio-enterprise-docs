# 2. Create New Assistant

Click **+ Add Assistant** in the top right to open the initialization modal. This is the first step to bring an assistant to life.

![](/assets/images/zh/admin/assistant-management-create-new_zh.png)

#### Basic Parameters

<table><thead><tr><th width="142.01171875">Field</th><th>Description</th><th>Example</th></tr></thead><tbody><tr><td><strong>Emoji</strong></td><td>Visual identifier for the assistant; choose by function for quick recognition.</td><td>💻 (Code assistant), ⚖️ (Legal assistant)</td></tr><tr><td><strong>Name *</strong></td><td>Unique identifier. Recommended format: <code>[Department/Function] + Role</code>.</td><td><code>R&D - Code Reviewer</code></td></tr><tr><td><strong>Model *</strong></td><td>Base model that determines intelligence. Supports cloud or local models.</td><td><code>anthropic/claude-3-5-sonnet</code></td></tr><tr><td><strong>Prompt (System Prompt) *</strong></td><td>Core instructions defining goals, output format, and constraints.</td><td><code>You are a senior Python engineer. Only output code, no explanations...</code></td></tr></tbody></table>

<Callout type="warn">
**Note:** The initial prompt is a draft. After creation, refine and test it in **Detailed Settings**.
</Callout>
