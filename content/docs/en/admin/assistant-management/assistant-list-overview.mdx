---
title: "Assistant List Overview"
description: View all configured AI assistants and monitor the distribution of knowledge base and MCP tool capabilities.
---

On the **Assistant Management** page, you can see all configured assistants. This panel helps monitor capability distribution.

<img src="/assets/images/en/admin/assistant-management-list-overview_en.webp" alt="" />
* **Search bar:** Search by assistant name.
* **Capability indicators:**
  * **Knowledge Base (KB):** Number of attached private knowledge bases.
  * **MCP:** Number of enabled tool services.
* **Actions:** **Edit** or **Delete** assistants.
