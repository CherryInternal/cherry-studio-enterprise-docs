An **Assistant** is the core unit of Cherry Studio Enterprise. Unlike generic chat, an assistant is a virtual employee pre-configured with a **role prompt**, **knowledge boundaries (Knowledge Base)**, and **action capabilities (MCP Tools)**.

With this module, admins can mass-produce standardized AI productivity tools and distribute them to user groups.

<Callout type="info">
**Core value:**

* **Out of the box:** Employees can use expert assistants without learning prompt engineering.
* **Knowledge augmentation:** Attach private enterprise documents (PDF/Word/Markdown) to enable precise Q&A.
* **Capability expansion:** Use MCP to enable web search, file read/write, or database queries.
</Callout>
