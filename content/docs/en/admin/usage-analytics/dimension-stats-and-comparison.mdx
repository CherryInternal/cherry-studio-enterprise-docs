title: "Dimension Stats and Comparison"
---

This module lets admins switch between **Users** and **Groups** for side-by-side comparison.

<img src="/assets/images/en/admin/usage-analytics-dimension-comparison_en.webp" alt="" />
### How to Use

Click **By User / By Group** in the top right to switch.

* **By Group:** Shows resource usage by department. For example, R&D usually consumes more tokens than admin.
* **By User:** Quickly identify power users. Blue is input tokens, green is output tokens, showing whether users read long inputs or generate long outputs.
