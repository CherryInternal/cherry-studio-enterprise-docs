The leaderboard is a quick list for identifying heavy users.

<img src="/assets/images/en/admin/usage-analytics-leaderboard_en.webp" alt="" />
## List Fields

* **Rank:** Sorted by consumption descending.
* **Name:** User nickname.
* **Group:** User department.
* **Requests:** User activity frequency.
* **Token usage:** Total resource consumption.

<Callout type="info">
**Best practice:** Review the Top 5 regularly. If usage is normal, ensure sufficient quota; if it spikes, investigate to prevent API key abuse.
</Callout>
