## 2. Core Modules

### 👥 User and Permission Management

Provides fine-grained user and permission controls to ensure enterprise security and compliance.

- **User Management**: Bulk import and manage employee accounts, set access permissions and usage quotas, monitor usage and activity, support multiple authentication methods (local auth, SSO)
- **Group Management**: Create and manage departments or project groups, batch-assign group permissions and resources, flexible org structure management

### 🤖 AI Service and Model Management

Centralized management of enterprise AI models for flexible usage and cost control.

- **Provider Management**: Built-in support for 40+ AI providers (OpenAI, Azure OpenAI, Google Gemini, Anthropic Claude, Baidu ERNIE, Alibaba Tongyi, etc.), with custom provider support for OpenAI / Gemini / Claude API formats
- **Default Models**: Set platform default assistant model, topic naming model, and translation model, configure usage quotas and rate limits per model

### 📋 Assistant Management

Manage enterprise AI assistants to provide users with pre-configured conversational assistants.

- **Assistant List**: Create and manage enterprise AI assistants, set visibility and group permissions
- **Advanced Configuration**: Prompt settings and visibility control, model selection and parameter tuning, knowledge base and MCP tool binding, assistant publishing and validation

### 🎭 Agent Management

Manage enterprise AI agents with more powerful autonomous task execution capabilities.

- **Agent Creation and Configuration**: Visual creation wizard, custom persona and behavior rules, configure available tools/APIs and knowledge base associations
- **Agent Marketplace**: Internal enterprise agent sharing and template management

### 📦 Resource Library

Centralized management of enterprise AI resources, containing five modules:

- **Agent Skills**: Manage skill modules available to agents, support manual upload or marketplace download, control availability by group
- **Assistant Templates**: Manage reusable assistant configuration templates, import from marketplace or create manually
- **Knowledge Base**: Multi-source data ingestion (PDF, DOCX, TXT, etc.), automatic parsing and smart chunking, vectorized retrieval with reranking
- **Mini Programs**: Add and manage mini program apps, configure name, icon, URL, and permissions
- **MCP Servers**: Configure enterprise MCP Server endpoints and tools, set access permissions, auto-distribute configs by user permissions

### 📊 Usage Analytics

Comprehensive data analysis and usage monitoring to help administrators understand platform usage.

- **Usage Overview**: Token consumption and request count statistics, multi-dimensional analysis by user, model, and time period, trend and distribution visualization charts
- **Leaderboard**: User usage rankings, sort by request count and token consumption, filter by group

### ⚙️ License Management and System Settings

- **License Management**: View license status and expiry, monitor licensed user count, license activation, update, and renewal
- **System Settings**: System configuration and parameter management, security policy settings, data backup and restore

### ℹ️ About

- **Application Information**: View system version and build information, client and Docker image download links
- **Release Notes**: View update details and release notes for each version
- **Technical Support**: Access official support contacts and service channels
