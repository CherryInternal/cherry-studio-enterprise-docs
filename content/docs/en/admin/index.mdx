The Cherry Studio Admin Console is a centralized management platform designed for enterprise users, offering comprehensive AI service management, user permission control, and resource configuration.

- **Console URL**: https://admin.demo.cherry-ai.com/
- **Demo account**: `admin`
- **Demo password**: `password`

## Core Modules

### 👥 User and Permission Management

Provides fine-grained user and permission controls to ensure enterprise security and compliance.

#### Employee Management

- Bulk import and manage employee accounts
- Set access permissions and usage quotas
- Monitor usage and activity
- Support multiple authentication methods (local auth, SSO)

#### Group Management

- Create and manage departments or project groups
- Batch-assign group permissions and resources
- Flexible org structure management

#### Access Control

- Role-based access control (RBAC)
- Fine-grained feature permissions
- API access permission management

### 🤖 AI Service and Model Management

Centralized management of enterprise AI models for flexible usage and cost control.

#### Provider Management (Model Access)

**Built-in support for 40+ AI providers**, including:

- OpenAI (GPT-4, GPT-3.5, etc.)
- Azure OpenAI
- Google Gemini
- Anthropic Claude
- Major domestic providers (Baidu ERNIE, Alibaba Tongyi, iFlytek Spark, etc.)

#### Custom Provider Configuration

Supports multiple API formats:

- OpenAI Chat Completions API format
- OpenAI Response API format
- Google Gemini API format
- Anthropic Claude API format

#### Model Policy Configuration

- **Default assistant model**: default model for common chat features
- **Topic naming model**: model used to auto-generate summaries/titles
- **Translation model**: default model for client translation
- **Model usage limits**: quotas and rate limits per model
- **Cost management**: configure usage cost and budget control

### 🎭 Agent Management

#### Agent Creation and Configuration

- Visual agent creation wizard
- Custom agent persona and behavior rules
- Configure tools and APIs available to agents
- Set knowledge base associations

#### Agent Marketplace

- Internal enterprise agent sharing
- Agent template management
- Usage statistics and effectiveness analysis

### 📚 Enterprise Knowledge Base Management

#### Knowledge Base Creation and Management

- **Multi-source data ingestion**:
  - Document uploads (PDF, DOCX, TXT, MD, CSV, etc.)
  - Web content crawling
  - API data ingestion
  - Database connections
- **Smart processing**:
  - Automatic document parsing and structuring
  - Smart chunking and index optimization
  - Multilingual support

#### Vectorization and Storage

- **Vector embeddings**: generate high-quality vectors with advanced embedding models
- **Vector database**: built-in high-performance vector DB with external DB support
- **Retrieval optimization**: hybrid search, reranking, and more

#### Knowledge Base Testing and Optimization

- Retrieval evaluation tools
- Answer quality assessment
- Knowledge base updates and version management

### 🔗 MCP Server Management

#### MCP Server Configuration

- Configure enterprise MCP Server endpoints
- Manage MCP tools and resources
- Set access permissions and security policies

#### Configuration Distribution

- Auto-distribute configs based on user permissions
- Support hot updates
- Configuration version management

### 📱 Mini Program Extension Management

#### Mini Program Configuration

- Add and manage mini program apps
- Configure app name, icon, and URL
- Set required parameters and permissions

#### App Integrations

- Third-party app integration
- OAuth configuration
- API gateway management

### ⚙️ System Management and Maintenance

#### License Management

- View license status and expiry
- Monitor licensed user count usage
- License renewal and upgrades

#### System Monitoring

- Real-time system performance monitoring
- API call statistics and analysis
- Error log and alert management

#### Data Management

- Data backup and restore
- Data import/export
- Data cleanup and archiving policies
