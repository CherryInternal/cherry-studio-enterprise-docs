## Download and Sign In

- **Client download**: https://enterprise.cherryai.com.cn/download

<img src="/assets/images/en/client-login-express_en.webp" alt="Client Login — Cloud Service" />

The client signs in with a cloud account using one of these options:

- **Login with SMS code**: Enter your phone number, receive a verification code, and log in.
- **Login with password**: Log in with your registered phone number and password.

If you don't have an account yet, click "Register" in the bottom-left corner.

<PremiumOnly>

### Self-Hosted Login

<img src="/assets/images/en/client-login-premium_en.webp" alt="Premium Login — Private Deployment" />

For self-hosted deployments, fill in the following:

- **Server URL**: Your organization's self-hosted server URL.
- **Username**: Username assigned by your administrator.
- **Password**: Your login password.

If SSO is configured, you can also click "SSO Login" in the bottom-left corner.

<Callout type="info">
**Demo account:** Server URL `https://api.demo.cherry-ai.com`, Username `user`, Password `password`
</Callout>

Click "Private Deployment" or "Cloud Service" in the bottom-right corner of the login page to switch between login modes.

</PremiumOnly>

## Runtime Dependencies

If you plan to use **agents**, the following runtimes must be installed on the computer running the client; they are not needed if you only use chat, knowledge base, and similar features.

**Match the list to your OS:**

- **Windows users**: install **Python**, **Node.js**, and **Git for Windows**
- **macOS / Linux users**: install **Python** and **Node.js**

### Python

Download page: https://www.python.org/downloads/

The page auto-detects your OS — click the large download button at the top to get the matching installer. The screenshot below is the macOS view; on Windows the button reads "Download the latest version for Windows".

<img src="/assets/images/shared/install/python-download.webp" alt="Python download page" />

After the download finishes, double-click the installer and follow the wizard — clicking **Continue** / **Next** through each step is enough to complete the install:

<img src="/assets/images/shared/install/python-installer.webp" alt="Python install wizard" style={{width:'60%'}} />

### Node.js

1. Open https://nodejs.org/ and click the **Get Node.js** button:

   <img src="/assets/images/shared/install/nodejs-home.webp" alt="Node.js homepage" />

2. On the download page, pick your OS from the dropdown at the top (Windows / macOS / Linux), then click the **Installer** button below to download the installer:

   <img src="/assets/images/shared/install/nodejs-download.webp" alt="Node.js download page" />

### Git for Windows (Windows only)

macOS / Linux ship with bash — skip this section.

Download page: https://git-scm.com/downloads/win

Click the **Click here to download** link at the top of the page to grab the latest installer. Keep the default options during install; **Git Bash** is included automatically.

<img src="/assets/images/shared/install/git-windows-download.webp" alt="Git for Windows download page" />
