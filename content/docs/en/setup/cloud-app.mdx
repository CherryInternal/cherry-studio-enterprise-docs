## 🚀 Quick Start

Cherry Studio Enterprise is now available in the Tencent Cloud Marketplace with one-click deployment, allowing you to experience the full enterprise AI platform without complex configuration.

<Callout type="warn">
Currently, one-click cloud app deployment is only supported on Tencent Cloud. We are actively expanding support for other cloud providers.
</Callout>

**Tencent Cloud Marketplace link**: [https://app.cloud.tencent.com/detail/SPU_BHFHEJFDEE4965](https://app.cloud.tencent.com/detail/SPU_BHFHEJFDEE4965)

<Callout type="info">
The Tencent Cloud Marketplace listing shows "Standard Edition", but the actual edition is determined by the **License** you receive upon purchase. The License supports both **Premium** and **Express** editions — the specific edition is based on your offline contract terms.
</Callout>

## 📋 Deployment Flow

### Step 1: Start Installation

1. Visit the [Tencent Cloud Cherry Studio installation page](https://app.cloud.tencent.com/detail/SPU_BHFHEJFDEE4965)
2. Sign in with your Tencent Cloud account
3. Click **Install** to select resources
4. Choose the nearest region for deployment

<Callout type="info">
    No servers available in the selected region? <a href="#no-machine">See solution</a>
</Callout>

### Step 2: Configure Deployment Parameters

The system will automatically detect your network environment. If you already have suitable network and security group settings, proceed to the next step.

<img src="/assets/images/zh/cloudappinstall/17579059506041.webp" alt="Generate app" />
### Step 3: Wait for Creation

App creation takes a few minutes. Please wait.

<img src="/assets/images/zh/cloudappinstall/17579075417655.webp" alt="Creation in progress" />
### Step 4: Get the Access Entry

After creation, open the app settings page to get the access entry:

<img src="/assets/images/zh/cloudappinstall/17579075847068.webp" alt="App settings entry" />
Click **Admin Console Entry (Public)** to access the Cherry Studio Enterprise admin console.

<Callout type="info">
    Can't access the admin console? The security group may not allow the request. <a href="#set-acl">See solution</a>
</Callout>

### Step 5: First Login

Use the **Cherry Studio Enterprise admin account** shown on the app settings page:

- Initial username: see the app settings page
- Initial password: see the app settings page

<img src="/assets/images/zh/cloudappinstall/17579076770749.webp" alt="Admin console entry" style={{width: '80%'}} />
### Step 6: Change the Default Password

**Strongly recommended** to change the default admin password after first login:

<img src="/assets/images/en/setup/change-password-1_en.webp" alt="User Management - Change Password Entry" />
<img src="/assets/images/en/setup/change-password-2_en.webp" alt="Change Password Dialog" />

## ❓ FAQ

### Q: What if there is no machine list when selecting an availability zone? [#no-machine]

A: Contact Tencent Cloud support to ask which zones have available resources. For example, Guangzhou 6/7 may have capacity while zone 2 may not. You can create a new VPC and subnet in a zone with available machines.

<Callout type="info">
    <a href="#create-vpc">How to create a VPC and subnet</a>
</Callout>

### Q: What if I cannot access the service after deployment?

A: Check the following:

1. Make sure the security group ports are correctly configured (36700, 36800, 32811)
2. Check network connectivity
3. Confirm you are using the correct access address and port

<Callout type="info">
    <a href="#set-acl">How to configure the security group</a>
</Callout>
