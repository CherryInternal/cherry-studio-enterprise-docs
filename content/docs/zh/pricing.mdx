## 版本选择指南

### 企业速通版（Express） vs 企业旗舰版（Premium）：关键差异

**功能完全相同**：企业速通版和企业旗舰版在功能上完全一致，都包含完整的企业级功能。

**核心差异仅有两点**：

1. **服务商配置方式**

   - 企业速通版：只能使用内置服务商，开箱即用，无需配置
   - 企业旗舰版：完全自由配置，可添加自定义服务商和私有化模型

2. **定价模式**
   - 企业速通版：¥9,999，不限制人数
   - 企业旗舰版：[联系我们以获取定价](/zh/docs/contact)

### 如何选择？

- **选择企业速通版**：希望快速部署，使用主流大模型服务，无需自定义服务商
- **选择企业旗舰版**：需要对接私有化模型、自建服务或有特殊服务商需求

## 各版本详细说明

### 社区版（Community）

---

**适用对象**：个人用户、开发者<br />
**部署方式**：单机使用<br />
**技术支持**：社区支持<br />
**许可协议**：AGPL 3.0 开源协议

**核心特性**：

- 完全免费，无使用时间限制
- 支持主流 AI 模型的基础对话功能
- 本地部署，数据完全自主控制
- 内置 30+ 服务商（OpenAI、Anthropic、Google Gemini、国产大模型等）
- **功能丰富但需手动配置**：用户需自行申请和配置各服务商的 API Key

**服务商使用方式**：

- 提供完整的服务商列表，但需用户自行注册账号
- 需要手动填写 API Key、Base URL 等配置信息
- 适合有技术背景、愿意自行管理 API 密钥的用户

**使用场景**：

- 个人 AI 助手使用
- 开发者学习和试用
- 小团队内部工具
- 对成本敏感的个人用户

### 企业速通版（Express）

---

**适用对象**：初创公司、中小型团队<br />
**部署方式**：私有化部署<br />
**技术支持**：专属技术支持（需购买年度服务费）<br />
**定价**：¥9,999，不限制人数

**核心特性**（与企业旗舰版功能完全一致）：

- ✅ 基础功能：完整的 AI 对话和多模型支持
- ✅ 服务商管理：内置主流大模型服务（不可自定义）
- ✅ 模型管理：管理和配置可用的 AI 模型
- ✅ 员工管理：企业员工账号管理和组织架构
- ✅ 共享知识库：企业级向量知识库和文档管理
- ✅ 权限控制：基于角色的访问控制（RBAC）
- ✅ 数据备份：完整的数据备份和恢复机制
- ✅ 助手管理：自定义 AI 助手配置和管理
- ✅ 智能体管理：企业智能体工作流编排
- ✅ MCP 管理：MCP 工具集成和管理
- ✅ 小程序管理：企业小程序生态支持

**服务商使用方式**：

- **使用内置服务商**：系统预配置好国内外主流大模型服务，**开箱即用**
- **不能自定义服务商**：不支持添加自己的 API Key 或其他服务商
- 无需复杂的 API Key 配置，适合希望快速上手的企业

**服务商限制**：

- 仅能使用系统内置的服务商和模型
- 无法添加私有化部署的模型服务
- 无法使用自己申请的 API Key

**使用场景**：

- 希望快速部署，开箱即用的中小企业
- 不需要自定义服务商配置的团队
- 对功能有完整需求但对服务商无特殊要求的企业
- 人数规模不受限制的团队
