# 管理后台配置

部署完成后，您需要在管理后台进行初始配置。

## 登录管理后台

1. 打开浏览器访问 `http://your-server:3680`
2. 使用默认管理员账号登录：
   - 用户名：`admin`
   - 密码：`admin123`（首次登录后请立即修改）

## 必要配置步骤

### 1. 配置服务商

添加 AI 模型服务商，让用户可以使用 AI 能力：

1. 进入 **服务商管理**
2. 点击 **添加服务商**
3. 选择服务商类型（如 OpenAI、Anthropic、Azure 等）
4. 填入 API Key 和配置信息
5. 测试连接并保存

详细文档：[服务商管理](/zh/admin/service-provider)

### 2. 创建分组

分组用于控制用户可以访问的模型资源：

1. 进入 **分组管理**
2. 点击 **创建分组**
3. 设置分组名称（如"研发组"、"市场组"）
4. 选择该分组可使用的模型

详细文档：[分组管理](/zh/admin/group-management)

### 3. 添加用户

为团队成员创建账号：

1. 进入 **用户管理**
2. 点击 **添加用户**
3. 填写用户信息（用户名、邮箱、密码）
4. 分配到对应的分组

详细文档：[用户管理](/zh/admin/user-management)

## 下一步

完成后台配置后，请继续 [客户端配置](/zh/quickstart/client-setup)。
