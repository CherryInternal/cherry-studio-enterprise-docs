# 发送第一条消息

恭喜！您已经完成了所有配置，现在来发送第一条消息验证一切正常。

## 开始对话

1. 在客户端中，点击 **新建对话**
2. 选择一个可用的 AI 模型
3. 在输入框中输入您的第一条消息，例如：

```
你好！请介绍一下你自己。
```

4. 按回车发送

## 验证成功

如果您收到了 AI 的回复，说明配置全部正确！

## 常见问题

### 无法连接服务器

- 检查服务器地址是否正确
- 确认服务已启动（访问 `http://your-server:3670/health`）
- 检查防火墙设置

### 没有可用的模型

- 确认管理员已配置服务商
- 确认您的用户已分配到正确的分组
- 联系管理员检查分组的模型权限

### 发送消息报错

- 检查服务商 API Key 是否有效
- 确认模型配额未用完
- 查看管理后台的调用日志
