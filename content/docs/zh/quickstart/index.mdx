# 快速开始

本指南将帮助您快速完成 Cherry Studio 企业版的部署和配置，从购买产品到发送第一条 AI 消息。

## 流程概览

```
购买授权 → 部署服务 → 后台配置 → 客户端配置 → 发送消息
```

整个流程大约需要 **15-30 分钟**，具体取决于您选择的部署方式。
