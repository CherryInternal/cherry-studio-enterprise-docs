**用户管理**是 Cherry Studio 企业版管理后台的指挥中心。作为系统管理员，您将在此模块对企业成员的数字身份进行统一的配置与监管。

该模块不仅是简单的账号增删改查工具，更是**资源分配**与**安全控制**的基石。通过将用户关联至特定的"分组"，您可以精确控制不同部门或职级的员工能够访问哪些 AI 模型（Model Providers）和助手资源，从而有效管理企业算力成本与数据权限。

### 核心功能

在此模块中，您可以执行以下关键任务：

* **全生命周期管理**：支持从账号创建、信息变更（邮箱、密码、手机号）到账号注销/删除的完整流程。
* **精细化权限分配**：通过**分组 (Groups)** 机制，将用户与特定的模型资源池绑定，实现"研发组用 Claude，市场组用 GPT-4"的差异化配置。
* **安全阻断**：支持一键切换用户的**激活/未激活**状态，在员工离职或账号异常时，能够毫秒级阻断其登录与 API 调用权限。
* **API 网关凭证**：为开发者或自动化脚本生成独立的 **API Key**，使其无需通过客户端即可调用 Cherry Studio 集成的模型能力。
* **多维度审计**：提供直观的列表视图，支持通过用户名检索，并可按账号状态或分组进行筛选，快速定位目标用户。

### 快速开始

请在管理后台左侧导航栏选择 **用户管理** 进入主界面。

<Callout type="warn">
**重要提示：** 用户创建后的**权限范围**完全取决于其所属的 **分组**。在添加用户之前，建议您先在"分组管理"模块中规划好企业的组织架构及对应的模型授权策略。
</Callout>
