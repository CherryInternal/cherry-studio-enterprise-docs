# 批量导入用户

当企业需要一次性创建大量用户账号时，逐个手动添加效率较低。批量导入功能支持通过 **Excel (.xlsx)** 或 **CSV (.csv)** 文件一次性导入多个用户，同时自动创建不存在的分组。

<Callout type="info">
**入口：** 用户管理页面右上角 **导入** 按钮，或分组管理页面右上角 **导入** 按钮。
</Callout>

## 操作流程

导入过程分为 3 个步骤：

### 步骤 1：上传文件

<img src="/assets/images/zh/admin/user-management-import-step1_zh.webp" alt="批量导入 - 上传文件" />

1. 点击 **下载 Excel 模板** 或 **下载 CSV 模板** 获取标准模板文件。
2. 按模板格式填写用户数据。
3. 将填写好的文件拖拽到上传区域，或点击选择文件。
4. 点击 **下一步** 进入数据预览。

### 步骤 2：数据预览

系统解析文件后展示预览表格，可在此检查数据是否正确。如有格式错误或缺失必填项，系统会高亮提示。

### 步骤 3：执行导入

确认无误后点击执行，系统批量创建用户和分组。导入完成后展示结果摘要（成功数、跳过数、失败数）。

<Callout type="warn">
**注意事项：**
* 模板第一行为表头，请勿修改。
* 若用户名已存在，可选择跳过或更新该用户信息。
* 分组不存在时会自动创建（仅管理员有此权限）。
* 建议首次导入前先用少量数据测试，确认格式无误后再批量操作。
</Callout>
