## 配置步骤

1. 导航至 **系统设置** > **网络设置**。
2. 确保企业代理服务器/网关已正常运行（如内网需经统一出口访问外部 API 的场景）。
3. 在下方输入框中填入代理地址。

## 配置项说明

<table><thead><tr><th width="122.96875">参数</th><th width="300.38671875">描述</th><th>示例值</th></tr></thead><tbody><tr><td><strong>系统代理</strong></td><td>全局 HTTP/HTTPS 代理地址。Cherry Studio 的核心对话、模型列表更新均通过此通道。</td><td><code>{'http://127.0.0.1:7890'}</code></td></tr><tr><td><strong>小程序代理</strong></td><td>(可选) 专为 Cherry Studio 内置小程序或插件服务的代理地址。通常与系统代理保持一致。</td><td><code>{'http://127.0.0.1:7890'}</code></td></tr></tbody></table>

## 代理端口参考

不同代理软件/网关的监听端口可能不同，请以您实际代理服务设置界面中的"本地监听端口"为准，并填入对应的 `http://<地址>:<端口>`。

<Callout type="warn">
**注意**：

1. 请勿填写 `socks5://` 协议，Cherry Studio 目前主要支持 `http://` 协议代理。
2. 配置完成后，建议去 **模型管理** 页面点击 **Check** 按钮验证连通性。
</Callout>
