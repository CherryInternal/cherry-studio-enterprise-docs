# 1. 基础设置

### 配置说明

1. 进入 **系统设置** > **基础设置**。
2. 根据实际需求调整同步参数。

<img src="/assets/images/zh/admin/system-settings-basic_zh.webp" alt="" />
### 参数详解

<table><thead><tr><th width="115.4765625">参数项</th><th width="219.5859375">描述</th><th>推荐值/操作</th></tr></thead><tbody><tr><td><strong>客户数据同步间隔</strong></td><td>控制客户端本地数据（如对话记录、设置）自动同步到云端/服务器的频率。单位为分钟。</td><td><strong>默认值：1</strong><br />• <strong>低频 (5-10)</strong>：适合低配设备，减少后台资源占用。<br />• <strong>高频 (1)</strong>：最大限度防止突发断电导致的数据丢失。</td></tr></tbody></table>

<Callout type="info">
**提示**：修改此设置后，应用会自动保存更改。如果您在网络环境较差的情况下使用，建议适当延长同步间隔。
</Callout>
