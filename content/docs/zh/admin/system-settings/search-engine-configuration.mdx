# 3. 搜索引擎配置

#### 支持的服务商

Cherry Studio 聚合了多种主流搜索 API，您可以根据需求选择：

* **Tavily** (推荐)：专为 AI 优化，检索结果干扰少，且能够直接提取页面内容。
* **智谱 (Zhipu)**：适合中文环境搜索。
* **SearxNG**：适合注重隐私的自建服务用户。
* **ExaSearch / Bocha**：其他神经搜索服务商。

#### 配置步骤

1. 在 **默认搜索引擎服务商** 下拉菜单中，选择您已获取 API Key 的服务商（例如 `Tavily`）。
2. 在下方对应的配置区域填入认证信息。

<img src="/assets/images/zh/admin/system-settings-search-engine_zh.webp" alt="" />
#### 参数详解

| 参数            | 描述                       | 获取/填写方式                                 |
| ------------- | ------------------------ | --------------------------------------- |
| **默认搜索引擎服务商** | **必选**。决定当前助手联网时实际调用的服务。 | 下拉选择，例如 `Tavily`。                       |
| **API Host**  | 服务商的接口地址。                | 通常保持默认即可。如果是自建服务（如 SearxNG），请输入您的服务器地址。 |
| **API Key**   | 您的私有访问密钥。                | 请前往对应服务商官网控制台获取。                        |
| **用户名**       | 仅 `SearxNG` 需要。          | 如果您的 SearxNG 设置了基础认证，请在此填写。             |

#### 验证与排查

配置完成后，创建一个开启了 **联网搜索** 开关的助手，并询问：“今天关于 AI 的最新新闻是什么？”。如果助手返回了带有链接引用的回答，即代表配置成功。
