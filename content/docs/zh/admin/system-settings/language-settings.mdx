---
title: "语言设置"
description: "设置管理后台的显示语言，支持自动检测、简体中文、英文和日文。"
---

## 设置方式

1. 进入 **系统设置** > **语言**。
2. 在 **选择语言** 下拉菜单中选择目标语言。

<img src="/assets/images/zh/admin/system-settings-language_zh.webp" alt="语言设置" />

## 可选语言

| 选项 | 说明 |
|---|---|
| **自动** | 根据浏览器语言自动匹配。 |
| **简体中文** | 管理后台显示为简体中文。 |
| **English** | 管理后台显示为英文。 |
| **日本語** | 管理后台显示为日文。 |

语言设置仅影响当前浏览器，修改后立即生效，无需刷新页面。
