## 2. 核心功能模块

### 👥 用户与权限管理

提供精细化的用户和权限管理体系，保障企业数据安全和合规运营。

- **用户管理**：批量导入和管理企业员工账号，设置访问权限和使用配额，监控使用情况和活跃度，支持多种认证方式（本地认证、SSO 单点登录）
- **分组管理**：创建和管理不同部门或项目组，批量分配组权限和资源，灵活的组织架构管理

### 🤖 AI 服务与模型管理

集中管理和配置企业可用的 AI 大模型，实现灵活调用与成本控制。

- **服务商管理**：内置支持 40+ AI 服务商（OpenAI、Azure OpenAI、Google Gemini、Anthropic Claude、百度文心、阿里通义等），支持 OpenAI / Gemini / Claude 等多种 API 格式的自定义服务商接入
- **默认模型**：设置平台默认助手模型、话题命名模型、翻译模型，配置模型使用配额和调用频率限制

### 📋 助手管理

管理企业内的 AI 助手，为用户提供预配置的对话助手。

- **助手列表**：创建和管理企业级 AI 助手，设置可见范围和分组权限
- **高级配置**：提示词设置与可见性控制、模型选择和参数调优、知识库和 MCP 工具绑定、助手发布与验证

### 🎭 智能体管理

管理企业内的 AI 智能体，提供更强大的自主任务执行能力。

- **智能体创建与配置**：可视化创建向导，自定义人设和行为规则，配置可用工具/API 和知识库关联
- **智能体市场**：企业内部智能体共享和模板管理

### 📦 资源库

集中管理企业的各类 AI 资源，包含五大模块：

- **智能体技能**：管理智能体可使用的技能模块，支持手动添加或从社区市场下载，按分组控制可用范围
- **助手模版**：管理可复用的助手配置模板，支持从模板市场导入或手动创建
- **知识库**：多源数据接入（PDF、DOCX、TXT 等）、自动解析与智能分块、向量化检索与重排序
- **小程序**：添加和管理小程序应用，配置名称、图标、访问 URL 和权限
- **MCP 服务器**：配置企业 MCP Server 地址和工具，设置访问权限，按用户权限自动下发配置

### 📊 用量统计

全面的数据分析与用量监控，帮助管理员了解平台使用情况。

- **使用概览**：Token 消耗量和请求次数统计，按用户、模型、时间维度的多维分析，趋势与分布可视化图表
- **排行榜**：用户使用量排行，按请求数和 Token 消耗量排序，支持按分组筛选

### ⚙️ 授权管理与系统设置

- **授权管理**：查看 License 状态和有效期，监控授权用户数，License 激活、更新和续期
- **系统设置**：系统基础配置和参数管理，安全策略设置，数据备份和恢复

### ℹ️ 关于我们

- **应用信息**：查看系统版本号和构建信息，客户端和 Docker 镜像下载链接
- **更新日志**：查看各版本的更新内容和发布说明
