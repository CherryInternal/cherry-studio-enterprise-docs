### Q1: 保存时提示 "JSON Parse Error"？

* **原因**：JSON 格式不标准。
* **解决**：检查是否在最后一项参数后多加了逗号，或者使用了中文引号。点击 **格式化 JSON** 按钮可自动修复简单错误。

### Q2: 助手调用工具时一直转圈或报错？

* **原因**：服务器环境缺少运行时依赖。
* **解决**：
  1. 确认运行 Cherry Studio 的主机上已安装 **Node.js** (对应 `npx`) 或 **Python**。
  2. 检查网络连接，首次运行 `npx` 可能需要从 npm 镜像源下载包，网络超时会导致启动失败。

### Q3: 如何确认 MCP 是否生效？

* **验证方法**：
  1. 进入 **助手管理**，将该 MCP 绑定到一个测试助手。
  2. 在对话框中输入需要调用工具的指令（如"读取 desktop 目录下的 test.txt"）。
