# MCP 服务器

![](/assets/images/zh/admin/mcp-server-overview_zh.png)

通过配置 MCP 服务器，您可以赋予 AI 助手以下能力：

* **联网能力**：通过浏览器工具实时搜索网页。
* **文件操作**：读取本地代码库或写入日志文件。
* **系统集成**：连接数据库、API 或企业内部系统。

<Callout type="info">
**核心优势**：一次配置，全员共享。管理员配置好的 MCP 服务器可以被分配给多个不同的助手使用，无需重复编写工具代码。
</Callout>
