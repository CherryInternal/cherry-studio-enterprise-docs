## 功能概要

批量导入分组允许管理员通过上传 Excel 或 CSV 文件一次性创建多个分组，适用于需要快速搭建组织架构的场景。

## 操作步骤

1. 进入管理后台 → 分组管理
2. 点击页面右上角的 **导入** 按钮
3. 在弹窗中下载标准模板（Excel 或 CSV 格式）
4. 按照模板规则填写分组数据
5. 上传填写好的文件，点击 **下一步** 开始导入

<img src="/assets/images/zh/admin/group-management-batch-import_zh.webp" alt="批量导入分组" />

### 模板字段说明

| 字段名 | 必填 | 说明 |
| --- | --- | --- |
| **name** | 是 | 分组名称，唯一标识 |
| **description** | 否 | 分组描述 |

## 注意事项

- 第一行为表头，请勿修改
- 已存在的分组名称可选择跳过或更新
