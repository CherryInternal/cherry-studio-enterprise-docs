# 默认模型设置

**路径：** 管理后台侧边栏 > **默认模型**

在此页面，管理员可以定义 Cherry Studio 企业版在不同业务场景下调用的**全局兜底模型**。合理的配置可以在保证用户体验（响应速度、回答质量）的同时，有效控制 Token 消耗成本。

![](/assets/images/zh/admin/default-model-settings-overview_zh.png)

<Callout type="info">
**配置前提**：此处的下拉列表仅显示您在 **\[模型管理]** 中已添加并启用的模型。如果您找不到想要的模型，请先前往模型管理页面进行添加。
</Callout>
