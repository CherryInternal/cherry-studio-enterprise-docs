为了帮助您快速决策，以下是针对不同企业类型的推荐配置清单：

## 🚀 方案 A：性能优先 (研发/算法团队)

* **默认助手**：`DeepSeek-Reasoner` (深度推理，解决复杂代码问题)
* **话题命名**：`Qwen-Turbo` (极速生成标题)
* **翻译模型**：`DeepSeek-V3.2`

## ⚖️ 方案 B：均衡性价比 (通用办公/行政)

* **默认助手**：`DeepSeek-V3.2` (兼顾速度与智能，成本仅为 R1 的几分之一)
* **话题命名**：`Qwen-2.5-7B` (本地或云端低成本模型)
* **翻译模型**：`DeepSeek-V3.2`

## 💰 方案 C：极速响应 (客服/销售查询)

* **默认助手**：`GPT-5.1-mini` 或 `DeepSeek-Lite` (秒回，无需等待)
* **话题命名**：`GPT-5.1-mini`
* **翻译模型**：`GPT-5.1-mini`

## 常见问题 (FAQ)

**Q: 如果我删除了这里正在使用的模型会发生什么？** A: 如果您在"模型管理"中删除了被设为默认的模型，系统可能会回退到内置的初始配置或报错。建议在删除模型前，先在此页面切换到其他替代模型。

**Q: 话题命名模型配置太差会怎样？** A: 如果选择了指令跟随能力极差的模型（如某些早期的 1B 小模型），它可能会把"请总结标题"这句话本身作为标题，或者生成出一串乱码，导致左侧历史记录列表难以阅读。
