---
title: "话题命名模型"
description: "配置自动将对话内容总结为简短标题的模型，建议使用轻量模型以降低成本。"
---

import { Callout } from 'fumadocs-ui/components/callout'

**定义**：当用户开启新对话并发送第一条消息后，系统会在后台默默调用此模型，将用户的长篇大论总结为一个简短的标题（例如将"帮我写一份关于周五会议的纪要..."总结为"周五会议纪要"）。

<Callout type="warn">
**⚠️ 成本与性能警示** 请**避免**在此处配置 **推理模型 (Reasoning Models)**（如 `DeepSeek-R1`, `OpenAI o1`）。

* **原因**：推理模型在输出前会进行长思维链思考（CoT），用于生成简单的标题属于"杀鸡用牛刀"。这不仅会导致标题生成**极度缓慢**，还会产生昂贵的、不必要的 Token 费用。
</Callout>

* **最佳实践**：
  * **推荐模型**：`Qwen 2.5 7B`, `GPT-5.1-mini`, `Gemini Flash`。
  * **核心需求**：**速度快、指令跟随性好、便宜**。它只需要能读懂"请总结为标题"这一条指令即可。
