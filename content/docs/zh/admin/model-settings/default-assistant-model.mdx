---
title: "默认助手模型"
---

**定义**：当用户创建一个新的对话或助手，且没有手动指定特定模型时，系统将自动使用该模型进行响应。

* **业务影响**：这是用户体验的第一印象。决定了用户"随手一问"时的智能程度。
* **配置建议**：
  * **追求体验**：推荐选择逻辑能力强的旗舰模型，如 `DeepSeek-R1` 或 `Gemini-3-Pro`。这能确保用户获得高质量的初始体验。
  * **追求性价比**：如果企业内部并发量大，建议选择"甜点级"模型（如 `DeepSeek-V3.2` 或 `Gemini-3-flash`），在速度和成本之间取得平衡。
