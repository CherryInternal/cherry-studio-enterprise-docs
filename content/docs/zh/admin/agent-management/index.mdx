# 智能体管理

**智能体 (Agent)** 是 Cherry Studio Enterprise 面向编程开发场景的高级 AI 能力。不同于"助手 (Assistant)"侧重预设提示词和知识库的对话交互，智能体具备**多模型协作**（主模型/规划模型/快速模型）、**权限模式控制**、**技能绑定**和 **MCP 工具集成**等能力，可以自主完成更复杂的多步骤任务。

通过此模块，管理员可以为企业团队创建和配置专业化的 AI 智能体，并按分组分发给不同的用户。

<Callout type="info">
**核心价值：**

* **多模型协作：** 为不同任务阶段配置最优模型 — 主模型负责核心推理，规划模型处理复杂分析，快速模型加速简单操作。
* **权限治理：** 管理员可精确控制智能体的默认权限模式和最高权限上限，保障企业安全策略。
* **技能扩展：** 从资源库绑定预置技能，赋予智能体代码分析、文件处理等专项能力。
* **MCP 工具集成：** 接入 MCP 服务器，让智能体调用外部工具和数据源。
</Callout>
