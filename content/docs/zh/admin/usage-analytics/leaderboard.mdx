排行榜是识别高频使用者的快速列表。

<img src="/assets/images/zh/admin/usage-analytics-leaderboard_zh.webp" alt="" />
### 列表字段

* **排名：** 按消耗量降序排列。
* **名称：** 用户昵称。
* **分组：** 用户所属部门。
* **请求数：** 该用户的活跃频次。
* **Token 用量：** 该用户的总资源消耗。

<Callout type="info">
**最佳实践：** 定期查看排行榜 Top 5 用户。如果是正常业务需求，应确保其配额充足；如果出现异常突增，需及时沟通以防止 API Key 滥用。
</Callout>
