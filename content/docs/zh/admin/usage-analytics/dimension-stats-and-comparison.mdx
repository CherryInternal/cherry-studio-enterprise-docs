# 4. 维度统计与对比

该模块允许管理员在**用户**与**分组**两个核心维度间切换，进行横向对比。

![](/assets/images/zh/admin/usage-analytics-dimension-comparison_zh.webp)

#### 操作方式

点击右上角的 **按用户 / 按分组** 切换视图。

* **按分组视图：** 直观展示各部门的资源占用。例如，研发部通常比行政部消耗更多 Token。
* **按用户视图：** 快速定位“超级用户”。蓝色代表输入 Token，绿色代表输出 Token，帮助判断用户是倾向于阅读长文（高输入）还是生成内容（高输出）。
