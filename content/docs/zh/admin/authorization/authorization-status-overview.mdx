# 1. 授权状态概览

进入 **授权管理** 页面，顶部面板是系统健康状况的“晴雨表”。

![导入授权](/assets/images/zh/admin/authorization-import_zh.webp)

![验证失败](/assets/images/zh/admin/authorization-validation-failed_zh.webp)

![系统已授权及运行正常](/assets/images/zh/admin/authorization-valid_zh.webp)

* **系统已授权 (License 有效)**：表示当前的 License 文件签名正确且在有效期内。
* **运行状态**：右侧的 ● 正常运行 指示器表明系统已成功连接到验证服务器（或离线校验通过），未发现篡改风险。

<Callout type="info">
**提示：** 建议管理员每月检查一次此状态。如果状态变为红色，系统可能会限制新建用户或暂停部分高级模型服务。
</Callout>
