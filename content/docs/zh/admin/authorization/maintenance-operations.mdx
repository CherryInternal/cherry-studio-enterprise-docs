# 3. 维护操作

在页面底部，您可以对授权文件进行更新或迁移操作。

#### 刷新授权信息

当您完成了续费或扩容（例如购买了更多的用户数）后，本地系统可能不会立即同步。

1. 点击底部的 **刷新信息** 按钮。
2. 系统将强制向授权服务器请求最新的 License 状态。
3. **成功标志：** 页面上的“过期时间”或“用户限制”数据更新为您最新购买的数值。

#### 撤销 License (迁移授权)

如果您需要更换服务器硬件，或者将生产环境迁移到新的私有云环境，您需要先释放当前的授权，才能在由于新机器上激活。

1. 点击 **刷新信息** 按钮旁边的 **┇ (更多选项)** 图标。
2. 在弹出的菜单中选择 **撤销 License**。
3. 在确认弹窗中输入确认信息（如要求）。

<Callout type="error">
**危险操作警告：**

**撤销 License** 会立即解除当前服务器的授权绑定！

* 操作后，系统将立即变为“未授权状态”。
* 所有非管理员用户可能无法登录。
* **除非您已准备好新的 License 文件或准备停用此服务器，否则请勿执行此操作。**
</Callout>

***

### 常见问题 (FAQ)

**Q: 为什么刷新信息失败？**&#x20;

A: 请检查服务器的网络连接。如果是纯内网环境（无法访问公网），自动刷新将无效。您需要联系技术支持获取新的离线 License 文件，并重新进行“导入”操作（通常在未授权状态下会出现导入入口）。

**Q: "用户限制" 满了怎么办？**&#x20;

A: 您有两个选择：

1. 前往 **用户管理**，删除或禁用不再使用的离职员工账号，释放名额。
2. 联系 Cherry Studio 销售代表购买额外的用户配额（User Seat）。

**Q: License 过期后数据会丢失吗？**&#x20;

A: **不会。** 授权过期仅限制服务的访问（如无法对话、无法登录），您的知识库、模型配置和历史对话数据依然保留在数据库中。一旦导入新的有效 License，系统即刻恢复正常。
