**入口路径：** 侧边栏 → 资源库 → 助手模版
</Callout>

**助手库**是 Cherry Studio 企业版的"大脑中枢"。在这里，企业管理员可以集中定义、存储和分发标准化的 AI 角色（Prompt）。通过预设的助手库，您可以确保所有员工在使用 AI 时，能够直接调用经过验证的高质量提示词，而无需每个人都成为 Prompt Engineer。

<img src="/assets/images/zh/admin/assistant-library-overview_zh.webp" alt="" />
### 核心价值

* **标准化交付：** 统一企业对外的 AI 话术风格（如：客服回复、代码规范）。
* **效率提升：** 员工无需从零编写提示词，点击即用。
* **知识复用：** 将内部专家的经验转化为 Prompt 模板，供全员共享。

<Callout type="info">
**提示：** 管理员在后台配置的助手，会实时同步到所有企业成员的 Cherry Studio 客户端"发现"或"助手"页面中。
</Callout>
