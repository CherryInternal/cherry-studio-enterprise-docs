# 3. 模板市场导入

为了帮助企业快速启动，Cherry Studio 内置了 100+ 经过优化的通用助手模板。

#### 导入流程

1.  点击右上角 **操作** > **从模板中添加**。

    <img src="/assets/images/zh/admin/assistant-library-template-market-1_zh.webp" alt="" />
2. **浏览/搜索：** 在弹出的模态框中，您可以浏览涵盖“职业”、“商业”、“工具”等多个领域的预设助手。
3.  **预览：** 查看模板的描述，确认是否符合需求。

    <img src="/assets/images/zh/admin/assistant-library-template-market-2_zh.webp" alt="" />
4. **添加：** 点击右侧的 **+ 添加** 按钮。

#### 后续调整

导入后的助手将出现在您的列表中。您可以对其进行**二次编辑**，例如：

* 修改“产品经理”助手的提示词，使其更符合您公司的产品开发流程。
* 调整“社群运营”助手的语气，使其符合您公司的品牌调性。

<Callout type="info">
**Tip:** 模板库是学习 Prompt 编写的最佳教材。您可以导入一个高质量模板，查看其 Prompt 写法，以此为基础进行修改。
</Callout>
