# 4. 分组策略

合理的分组能帮助员工更快找到所需的工具。

#### 分组逻辑

![分组选择器](/assets/images/zh/admin/assistant-library-group-selector_zh.png)

在创建或编辑助手时，**所属分组** 字段支持多选。

* **精选 (Featured)：** 建议将全公司最高频使用的通用助手放入此组。
* **职业 (Professional)：** 放置特定岗位专用助手（如：HR 面试官、Java 架构师）。
* **工具 (Tools)：** 放置功能型助手（如：翻译官、JSON 格式化工具）。

<Callout type="info">
**系统逻辑：** 这里的“分组”直接对应客户端助手页面的标签栏。如果一个助手未分配任何分组，用户可能需要在“全部”列表中翻阅才能找到，建议至少分配一个分组。
</Callout>
