在管理后台左侧导航栏中点击「订单管理」，即可查看所有订单记录。

<img src="/assets/images/zh/admin/order-management-list_zh.webp" alt="订单管理列表" />

## 列表字段说明

| 字段 | 说明 |
| --- | --- |
| **订单号** | 系统自动生成的唯一订单编号。 |
| **商品** | 购买的商品名称，如「Token 充值」或「Cherry Studio Enterprise 极速版」。 |
| **金额** | 订单金额。 |
| **状态** | 订单当前状态（详见下方状态说明）。 |
| **支付方式** | 对公转账或支付宝。 |
| **创建时间** | 订单创建的日期和时间。 |

## 搜索与筛选

- **搜索框**：支持按订单号或商品名称搜索。
- **状态筛选**：可按「全部状态」、「已完成」、「已关闭」等状态筛选订单。

## 订单状态说明

| 状态 | 说明 |
| --- | --- |
| **待支付** | 订单已创建，等待完成支付。 |
| **已支付** | 支付已完成，等待系统确认并开通服务。 |
| **已完成** | 支付确认完毕，服务已开通或 Token 已到账。 |
| **已关闭** | 订单已关闭（超时未支付或手动取消）。 |

<Callout type="info">
**待支付订单**会显示剩余支付时间倒计时，您可以点击「支付」继续付款，或点击「取消」关闭订单。
</Callout>
