---
title: "订单详情"
description: "查看单个订单的完整信息，包括订单信息和商品信息。"
---

点击某一个订单，可查看该订单的完整详情。

<img src="/assets/images/zh/admin/order-management-detail_zh.webp" alt="订单详情" />

## 订单信息

| 字段 | 说明 |
| --- | --- |
| **订单号** | 唯一订单编号。 |
| **企业名称** | 下单企业名称。 |
| **创建时间** | 订单创建时间。 |
| **支付时间** | 实际完成支付的时间。 |
| **支付方式** | 对公转账或支付宝。 |

## 商品信息

| 字段 | 说明 |
| --- | --- |
| **商品名称** | 购买的商品，如「Token 充值」。 |
| **商品编码** | 商品的系统编码，如 `token-recharge`。 |
