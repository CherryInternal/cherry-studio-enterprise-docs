# 助手管理

**助手 (Assistant)** 是 Cherry Studio Enterprise 的核心原子单元。不同于通用的 AI 对话，"助手"是预置了特定 **角色设定 (Prompt)**、**知识边界 (Knowledge Base)** 和 **行动能力 (MCP Tools)** 的虚拟员工。

通过此模块，管理员可以批量生产标准化的 AI 生产力工具，并将其分发给企业内的不同用户组。

<Callout type="info">
**核心价值：**

* **开箱即用：** 员工无需学习复杂的提示工程，直接使用管理员配置好的专家助手。
* **知识增强：** 将企业私有文档（PDF/Word/Markdown）挂载给助手，实现精准问答。
* **能力扩展：** 通过 MCP 协议，赋予助手联网搜索、文件读写或数据库查询的能力。
</Callout>
