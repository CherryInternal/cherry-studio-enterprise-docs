# 1. 助手列表概览

进入 **助手管理** 页面，您将看到企业内所有已配置的 AI 助手清单。此面板用于全局监控助手的能力分布。

![](/assets/images/zh/admin/assistant-management-list-overview_zh.png)

* **搜索栏：** 支持通过助手名称快速检索。
* **能力指标：**
  * **知识库 (KB)：** 显示该助手挂载了多少个私有知识库。
  * **MCP：** 显示该助手启用了多少个扩展工具服务。
* **操作：** 支持 **编辑** 配置或 **删除** 废弃助手。
