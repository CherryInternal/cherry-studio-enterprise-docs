# 2. 创建新助手

点击右上角的 **+ 添加助手** 按钮，系统将弹出初始化模态框。这是赋予助手“生命”的第一步。

![](/assets/images/zh/admin/assistant-management-create-new_zh.png)

#### 基础配置参数

<table><thead><tr><th width="142.01171875">参数</th><th>说明</th><th>示例</th></tr></thead><tbody><tr><td><strong>表情符号 (Emoji)</strong></td><td>助手的视觉标识，建议根据职能选择，便于在侧边栏快速识别。</td><td>💻 (代码助手), ⚖️ (法务助手)</td></tr><tr><td><strong>名称 *</strong></td><td>助手的唯一标识名。建议采用 <code>[部门/职能] + 角色</code> 的命名规范。</td><td><code>研发部-代码审查员</code></td></tr><tr><td><strong>模型 *</strong></td><td>决定助手智商的底座模型。支持选择云端模型或本地模型。</td><td><code>anthropic/claude-3-5-sonnet</code></td></tr><tr><td><strong>提示词 (System Prompt) *</strong></td><td>助手的核心指令。定义其任务目标、输出格式及行为禁忌。</td><td><code>你是一个资深 Python 工程师，请只输出代码，不要解释...</code></td></tr></tbody></table>

<Callout type="warn">
**注意：** 初始化时的提示词仅为草稿，创建后建议进入“详细设置”进行微调和测试。
</Callout>
