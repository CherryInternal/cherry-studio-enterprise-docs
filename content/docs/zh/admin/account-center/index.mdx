## 入口

点击管理后台**左下角的人物图标**，在弹出的菜单中选择**账号设置**，即可进入账号中心。

<img src="/assets/images/zh/admin/account-center-entry_zh.webp" alt="账号中心入口" />

<Callout type="info">
账号中心是**用户自助管理**入口，所有已登录的账号都可以访问，与管理员侧栏中的「订单管理」「用户管理」等功能不同。
</Callout>

## 功能概览

账号中心包含三个标签页：

| 标签页 | 说明 |
| --- | --- |
| **账号设置** | 查看个人信息（用户名、邮箱）和修改登录密码。 |
| **我的订单** | 查看与当前账号关联的所有订单记录。 |
| **关联组织** | 查看您加入或创建的所有组织，进入对应组织的管理后台，或通过邀请码加入新组织。 |

页面顶部还提供**下载客户端**入口，方便下载 Cherry Studio 客户端开始使用 AI 服务。
