在账号中心点击 **我的订单**，即可查看与当前账号关联的所有订单记录。

<img src="/assets/images/zh/admin/account-center-orders_zh.webp" alt="我的订单" />

## 列表字段说明

| 字段 | 说明 |
| --- | --- |
| **商品名称** | 购买的商品，如「Cherry Studio Enterprise 极速版」「Token 充值」。 |
| **订单号** | 系统自动生成的唯一订单编号。 |
| **关联组织** | 该订单归属的组织。 |
| **创建时间** | 订单创建的日期。 |
| **金额** | 订单金额。 |
| **状态** | 订单当前状态（已完成/待支付/已关闭等）。 |

## 搜索与筛选

- **搜索框**：支持按订单号、产品名或组织名快速搜索。
- **类型筛选**：按商品类型筛选（全部类型、套餐、Token 充值等）。
- **状态筛选**：按订单状态筛选。

<Callout type="info">
管理员视角的「订单管理」（侧栏入口）可查看和管理整个组织的全部订单。账号中心的「我的订单」只显示与当前登录账号相关的订单。
</Callout>
