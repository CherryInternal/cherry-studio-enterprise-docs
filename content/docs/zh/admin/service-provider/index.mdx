## 概览

Cherry Studio 企业版（速通版 / Express）为追求效率的个人和团队提供"零配置"的 AI 工作台体验。无需逐家注册账号、无需分别申请 API Key、也无需自行调试参数，即可一站式接入主流大模型，开箱即用。

## 核心特性

* **开箱即用**：下载即用，无需填写 API Key，无需逐家配置服务商。
* **一站式聚合**：内置接入 **DeepSeek**、**智谱 GLM**、**Kimi**、**MiniMax**、**千问Qwen** 等多家主流大模型，一个入口随心切换。
* **官方同价**：计费策略与模型原厂保持一致，**拒绝溢价**。用多少充多少，余额永久有效。
* **统一计费**：无需管理多个不同的订阅账号，一个账号余额通行所有模型。

## 适用人群

* 希望"零配置"立即开始工作的个人与团队。
* 不想逐家申请、维护多个模型账号与 API Key 的用户。
* 需要在多家主流模型之间灵活切换、按量付费的用户。
