## 概览

Cherry Studio 企业版（速通版 / Express）为追求效率的个人和团队提供"零配置"的 AI 工作台体验。无需注册各家大模型账号，无需申请海外信用卡，也无需配置网络环境，即可直接使用全球最顶尖的大模型。

## 核心特性

* **开箱即用**：下载即用，无需填写 API Key，无需配置代理地址。
* **全球主流模型**：内置支持 **DeepSeek-R1**、**智谱GLM-4**、**千问Qwen** 等顶级模型。
* **官方同价**：计费策略与模型原厂完全一致，**拒绝溢价**。用多少充多少，余额永久有效。
* **统一计费**：无需管理多个不同的订阅账号，一个账号余额通行所有模型。

## 适用人群

* 无法申请海外 API Key 的国内用户。
* 不想折腾网络配置，只想立即开始工作的文字工作者。
* 需要偶尔使用高级模型（如 DeepSeek-R1）但不想按月订阅的用户。
