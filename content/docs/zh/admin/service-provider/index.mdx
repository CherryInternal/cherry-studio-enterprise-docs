## 📚 版本概览&#x20;

Cherry Studio 致力于为不同类型的用户提供最佳的 AI 工作台体验。无论您是追求极致控制的技术极客、需要开箱即用的企业员工，还是对合规性有严格要求的政府机构，我们都提供了专属的解决方案。

目前，Cherry Studio 提供以下三个核心版本：

1. **速通版/Express (SaaS)**
2. **旗舰版/Premium (BYOK)**
3. **Qwen - 通义专版 (Aliyun)**

***

### 🚀 速通版/Express

> **关键词：** 开箱即用、全球模型、官方原价、无需魔法

速通版是为**追求效率的个人和团队**打造的"零配置"解决方案。您无需注册 ChatGPT 或 Claude 账号，无需申请复杂的海外信用卡，也无需配置网络环境，即可直接使用全球最顶尖的模型。

**核心特性**

* **开箱即用**：下载即用，无需填写 API Key，无需配置代理地址。
* **全球主流模型**：内置支持 **GPT-5**, **Claude 4.5 Sonnet**, **Google Gemini 3 Pro** 等顶级模型。
* **官方同价**：计费策略与模型原厂（如 OpenAI/Anthropic）完全一致，**拒绝溢价**。用多少充多少，余额永久有效。
* **统一计费**：无需管理 5 个不同的订阅账号（ChatGPT Plus, Claude Pro 等），一个账号余额通行所有模型。

**适用人群**

* 无法申请海外 API Key 的国内用户。
* 不想折腾网络配置，只想立即开始工作的文字工作者。
* 需要偶尔使用高级模型（如 Claude 4.5）但不想按月订阅的用户。

***

### 🛠️ 旗舰版/Premium (BYOK)

> **关键词：** 灵活性、自有 Key、本地模型、隐私优先

旗舰版是 Cherry Studio 的基石，专为**开发者、技术极客和已拥有 API 资源的企业**设计。它采用 **BYOK (Bring Your Own Key)** 模式，即"自带密钥"。

**核心特性**

* **全能连接**：支持配置 OpenAI、Azure、DeepSeek、Anthropic、Google Gemini 等任意兼容 OpenAI 协议的商业接口。
* **本地模型支持**：完美支持 Ollama、vLLM、LM Studio。您的数据完全在本地运行，无需上传云端，适合处理敏感数据。
* **极致自定义**：您可以精细控制每个模型的上下文长度、温度（Temperature）和系统提示词。

**适用人群**

* 拥有自己 API Key 的开发者。
* 需要在本地运行大模型（Local LLM）的用户。
* 希望将多个不同渠道的模型聚合到一个界面的超级个体。

***

### ☁️ Qwen 通义专版

> **关键词：** 阿里云百炼、国内合规、企业级稳定、深度集成

通义专版是与**阿里云百炼 (Aliyun Bailian)** 深度合作的企业级版本。系统预置了百炼的接入协议，专注于提供稳定、合规、高速的国内大模型服务。

**核心特性**

* **百炼生态集成**：默认接入通义千问全系列（Qwen-Max/Plus/Turbo）及百炼托管的 CodeQwen 模型。
* **企业级保障**：依托阿里云基础设施，提供高并发支持和企业级 SLA，数据传输符合国内安全合规要求。
* **极简部署**：只需填入一个阿里云 API Key，即可自动同步并管理企业可用的所有模型。

**适用人群**

* 对数据出境有严格限制的企业或政府机构。
* 阿里云百炼的现有企业客户。
* 主要依赖通义千问系列模型进行生产作业的团队。

***

### ⚖️ 版本对比指南

为了帮助您做出决定，请参考以下特性对比表：

| 特性维度     | 速通版/Express               | 旗舰版/Premium        | Qwen 通义专版         |
| -------- | --------------------------- | ------------------- | ----------------- |
| **核心理念** | 简单易用，全球直连                   | 自由连接，自带资源           | 国内合规，阿里生态         |
| **模型支持** | GPT-5, Claude Opus 4.6, Gemini 等 | 无限制 (取决于您拥有的 Key)   | Qwen 全系, 百炼托管开源模型 |
| **配置难度** | ⭐ (零配置)                     | ⭐⭐⭐ (需手动填 Key/Host) | ⭐⭐ (仅需填 Key)      |
| **网络要求** | **无需特殊网络** (内置加速)           | 用户自行解决网络通畅          | 国内直连 (速度极快)       |
| **计费方式** | **Cherry 统一钱包 (官方原价)**      | 直接支付给各模型服务商         | 阿里云百炼账单结算         |
| **本地模型** | ❌ 不支持                       | ✅ 支持 (Ollama 等)     | ✅ 支持 (Ollama 等)   |
| **推荐场景** | 个人办公、无需运维、尝鲜顶级模型            | 开发者、本地部署、多源聚合       | 国内企业落地、合规生产       |

**我该如何选择？**

* **想零配置直接使用全球顶级模型（GPT、Claude、Gemini），无需管理多个账号和网络环境** 👉 选择 **速通版/Express**。
* **已有 API Key 或本地模型，需要自由接入多个服务商并深度自定义** 👉 选择 **旗舰版/Premium**。
* **主力使用通义千问系列模型** 👉 选择 **Qwen 专版**。
