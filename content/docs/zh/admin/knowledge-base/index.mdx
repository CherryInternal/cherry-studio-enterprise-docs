# 知识库管理

**知识库 (Knowledge Base)** 是 Cherry Studio 企业版实现 **RAG (检索增强生成)** 技术的核心组件。它充当了大模型的“外挂大脑”，允许企业将私有的文档、网页、笔记等非公开数据转化为向量存储。

当用户与助手交互时，系统会先在知识库中精准检索相关信息，并将其作为上下文投喂给模型，从而实现基于企业事实的精准问答，显著减少“幻觉”并保障数据安全。
