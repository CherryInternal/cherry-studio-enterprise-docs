# 1. 界面概览与系统预设

进入 **小程序管理** 页面，您将看到当前企业环境内所有可用的应用列表。

#### 界面组成

<img src="/assets/images/zh/admin/mini-program-overview_zh.webp" alt="" />
列表分为两类应用：

1. **系统预设 (System):** Cherry Studio 官方精选的主流 AI 服务（如 DeepSeek, ChatGPT, 通义千问等）。此类应用由官方维护地址，管理员仅需控制开启/关闭。
2. **自定义 (Custom):** 由企业管理员手动添加的内部或第三方链接。

<Callout type="info">
**为什么有些应用无法删除？** 系统预设的小程序是硬编码在系统中的，为了保证服务的稳定性，仅支持**禁用**（隐藏），不支持物理删除。
</Callout>

#### 快速启用/禁用

在列表右侧的 **已启用** 列，通过开关即可控制该应用是否显示在员工的客户端侧边栏。

* **ON (开启):** 所有企业成员重启客户端后，均可在侧边栏看到该图标。
* **OFF (关闭):** 该应用将被隐藏，但配置保留。
