**路径：** **关于我们** > **更新日志**

此模块详细记录了 Cherry Studio 的演进历史，帮助管理员了解每次升级带来的具体变化。

<img src="/assets/images/zh/admin/about-us-release-notes_zh.webp" alt="" />
### 日志结构说明

系统按时间倒序排列所有历史版本，每个版本卡片包含以下信息：

* **版本号：** 如 `v0.4.10`。
* **发布日期：** 如 `2026/03/30`。
* **变更详情：** 具体的功能点描述。

<Callout type="info">
**最佳实践：** 在遇到系统异常（如数据统计不准、页面显示错位）时，建议先查看最近一次更新日志的 **修复** 类目，确认是否为已知问题并已在最新版中修复。
</Callout>
