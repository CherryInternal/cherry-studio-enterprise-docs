# 1. 应用信息

**路径：** **关于我们** > **应用信息**

此面板提供了当前 Cherry Studio 部署实例的核心运行指标和必要的资源下载入口。它是判断系统是否健康运行的首要仪表盘。

**注意：** 如果 **服务状态** 显示异常，虽然前端页面可能仍能访问，但模型调用、文件解析等核心后端功能将会失效。

### 资源下载中心

管理员可在此处快速获取配套工具，确保客户端与服务端的版本匹配：

* **客户端下载：** 提供 Windows 和 macOS 版本的 Cherry Studio Enterprise 客户端下载链接。
* **帮助文档：** 跳转至官方详细的操作手册（即您当前阅读的文档库）。
* **Docker 镜像：** 跳转至 Docker Hub 官方仓库，用于服务器端的升级维护操作。
