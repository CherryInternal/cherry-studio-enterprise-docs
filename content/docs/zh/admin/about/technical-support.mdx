# 3. 技术支持

**路径：** **关于我们** > **技术支持**

当文档无法解决您的问题，或遇到系统级故障时，请通过此页面联系 Cherry Studio 官方产研团队。

![](/assets/images/zh/admin/about-us-technical-support-1_zh.webp)

#### 提交工单流程

![](/assets/images/zh/admin/about-us-technical-support-2_zh.webp)

点击 **提交工单** 按钮，系统将跳转至企业级工单管理系统：

1. **身份认证：** 点击右上角登录（支持微信扫码登录）。
2. **填写表单：** 详细描述问题现象。
3. **上传凭证：** 建议附带 **应用信息** 页面的截图以及 **Docker 错误日志**。

#### 服务响应标准 (SLA)

官方团队承诺的响应时效如下，请根据业务紧急程度合理预期：

* **工作时间：** 周一至周五 10:00 - 18:00 (GMT+8)。
* **一般问题：** **7 个工作日** 内响应（如功能咨询、非阻塞性 Bug）。
* **紧急问题：** **24 小时** 内响应。

<Callout type="error">
**紧急问题的定义：** 仅当系统出现 **完全不可用**（如无法启动、全员无法登录）、**核心数据丢失** 或 **重大安全漏洞** 时，才被视为紧急问题。

**首次响应时间**：仅指技术人员**受理并回复工单的时间**，不代表问题解决时间。我们将尽商业上的合理努力解决问题，但不对具体修复时间作保证。
</Callout>
