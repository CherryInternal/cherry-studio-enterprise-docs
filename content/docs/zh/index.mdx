## 版本概览

Cherry Studio 企业版提供两个版本，您可以根据需要选择：

| | **Express 版**（云端托管） | **Premium 版**（私有化部署） |
| --- | --- | --- |
| **部署方式** | 无需部署，注册即用 | Docker / Helm 自行部署 |
| **数据存储** | Cherry Studio 云端托管 | 完全在您自己的服务器上 |
| **适合场景** | 快速上手、中小团队 | 数据合规要求高、大型组织 |
| **运维** | 由 Cherry Studio 维护 | 企业自行运维 |
| **计费方式** | 按套餐订阅（支持支付宝/对公转账） | 按授权许可证计费 |

<Callout type="info" title="如何选择？">
- 希望**零部署、即刻上手** → 选择 <DynamicLink href="/[lang]/docs/quickstart">Express 版快速开始</DynamicLink>
- 需要**数据完全私有、自主可控** → 选择 <DynamicLink href="/[lang]/docs/quickstart">Premium 版快速开始</DynamicLink>
</Callout>

## 数据安全与隔离

无论您选择 Express 版还是 Premium 版，Cherry Studio 企业版都为您的数据安全提供多层保障：

| 安全能力 | Express 版 | Premium 版 |
| --- | --- | --- |
| **数据隔离** | 基于分组的严格数据隔离，各组织数据完全独立，互不可见 | 独立部署，天然物理隔离 |
| **传输加密** | 全链路 HTTPS 加密 | 由企业自行配置 SSL/TLS |
| **身份认证** | JWT + OAuth2 认证，密码加密存储 | JWT 认证，密码加密存储 |
| **权限控制** | 基于角色的细粒度权限体系（管理员 / 组管理员 / 成员） | 同左 |
| **API 安全** | 独立 API Key，按用户隔离调用权限 | 同左 |

<Callout type="info" title="Express 版数据隔离机制">
Express 版采用共享基础设施 + 应用层隔离架构。每个组织的数据（模型配置、知识库、助手、文件等）通过严格的分组隔离机制互相隔离，任何 API 请求都会经过身份验证和权限校验，确保只能访问本组织的数据。
</Callout>
