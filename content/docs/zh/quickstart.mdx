## 第三步：添加服务商（仅旗舰版 Premium）

<Callout type="warn">
此步骤仅适用于 **旗舰版（Premium）**。Express 版已内置服务商，无需手动添加。
</Callout>

登录管理后台，进入「服务商管理」添加 AI 服务商（如 OpenAI、Anthropic 等），配置 API Key。

<Callout type="info">
  首次登录管理后台时，如需查看初始化管理员账号说明，请先阅读 <DynamicLink href="/[lang]/docs/setup">安装与部署文档</DynamicLink>；云应用部署请以应用设置页面提供的账号信息为准。
</Callout>

<Callout type="info">
  <DynamicLink href="/[lang]/docs/admin/service-provider">查看完整服务商配置文档 →</DynamicLink>
</Callout>
