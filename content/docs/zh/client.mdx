## 下载与登录

- **客户端下载**: https://enterprise.cherryai.com.cn/download

<img src="/assets/images/zh/client-login-express_zh.webp" alt="客户端登录 — 云服务" />

客户端使用云服务账号登录，支持以下两种方式：

- **验证码登录**：输入手机号，获取并输入验证码即可登录。
- **密码登录**：使用已注册的手机号和密码登录。

如果还没有账号，可点击左下角「注册账号」进行注册。

<PremiumOnly>

### 私有化部署登录

<img src="/assets/images/zh/client-login-premium_zh.webp" alt="Premium 版登录 — 私有化部署" />

私有化部署版本使用以下信息登录：

- **服务端地址**：企业自行部署的服务器 URL。
- **用户名**：由管理员分配的用户名。
- **密码**：对应的登录密码。

如已配置 SSO 单点登录，也可点击左下角「一键认证登录」。

<Callout type="info">
**体验账号：** 服务端地址 `https://api.demo.cherry-ai.com`，用户名 `user`，密码 `password`
</Callout>

点击登录页面右下角的「私有化部署」或「云服务」可在两种登录方式之间切换。

## 运行时依赖

如果您要使用**智能体**，需要在客户端所在电脑上预装下列运行时；仅使用智能对话、知识库等功能则无需安装。

**按系统对照您需要装的项目：**

- **Windows 用户**：需要 **Python**、**Node.js**、**Git for Windows**
- **macOS / Linux 用户**：需要 **Python**、**Node.js**

### Python

下载页面：https://www.python.org/downloads/

该页面会自动识别您的操作系统，点击页面顶部的下载按钮即可获取对应安装包。截图以 macOS 为例，Windows 用户打开时按钮会显示为 "Download the latest version for Windows"。

<img src="/assets/images/shared/install/python-download.webp" alt="Python 下载页面" />

下载完成后双击安装包运行，按照安装向导的提示一路点击 **Continue** / **下一步** 即可完成安装：

<img src="/assets/images/shared/install/python-installer.webp" alt="Python 安装向导" style={{width:'60%'}} />

### Node.js

1. 访问 https://nodejs.org/，点击 **Get Node.js** 按钮：

   <img src="/assets/images/shared/install/nodejs-home.webp" alt="Node.js 首页" />

2. 在下载页面顶部的下拉框中选择您的操作系统（Windows / macOS / Linux），然后点击下方的 **Installer** 按钮下载安装包：

   <img src="/assets/images/shared/install/nodejs-download.webp" alt="Node.js 下载页面" />

### Git for Windows（仅 Windows 用户）

macOS / Linux 自带 bash，可跳过本节。

下载页面：https://git-scm.com/downloads/win

打开后点击页面顶部的 **Click here to download** 链接下载最新版安装包。安装时保持默认选项即可，完成后 **Git Bash** 会随之安装好。

<img src="/assets/images/shared/install/git-windows-download.webp" alt="Git for Windows 下载页面" />

安装完成后，创建智能体时 **Git Bash** 路径会自动识别。如果路径为空（如左图），说明尚未安装 Git for Windows：

- 4GB RAM（建议8GB）
